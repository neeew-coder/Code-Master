JavaScript RegExp Methods

The RegExp object and string methods provide powerful tools for pattern matching, text extraction, and transformation.

🔍 RegExp exec()

const regex = /Code/;
const result = regex.exec("Welcome to CodeMaster!");
Output will appear here.

✅ RegExp test()

const regex = /Master/;
regex.test("Welcome to CodeMaster!");
Output will appear here.

📌 RegExp Properties

const regex = /code/gi;
regex.source; // "code"
regex.flags;  // "gi"
Output will appear here.

🔎 String match()

const text = "CodeMaster rocks!";
const result = text.match(/Code/g);
Output will appear here.

♻️ String replace()

const text = "Visit Microsoft!";
const result = text.replace(/Microsoft/, "CodeMaster");
Output will appear here.
