JavaScript Date Objects

JavaScript Date objects let you work with calendar dates and times. They are created using the new Date() constructor and can be formatted or manipulated using built-in methods.

📅 Current Date

const now = new Date();
▶️ Run Current Date
Output will appear here.

📆 From Date String

const d1 = new Date("2022-03-25");
▶️ Run Date String
Output will appear here.

📆 From Components

const d2 = new Date(2024, 10, 18); // Nov 18, 2024
▶️ Run Component Date
Output will appear here.

⏱️ From Milliseconds

const d3 = new Date(86400000); // Jan 2, 1970
▶️ Run Milliseconds Date
Output will appear here.

🌐 ISO Format

now.toISOString();
▶️ Run ISO Format
Output will appear here.

🧠 Notes

  • new Date() creates a date object from current time
  • new Date("YYYY-MM-DD") parses a date string
  • new Date(year, month, ...) uses components (month is 0-based)
  • new Date(ms) uses milliseconds since Jan 1, 1970
  • Use .toString(), .toDateString(), .toUTCString(), .toISOString() to format output
