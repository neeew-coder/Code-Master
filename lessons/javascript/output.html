JavaScript Output

JavaScript can display output in several ways — each suited for different tasks. You can write directly to the HTML, log to the console, show alerts, or update elements dynamically.

📘 What You’ll Learn

  • How to log messages to the console using console.log()
  • How to write directly to the document with document.write()
  • How to show popup messages using alert()
  • How to update page content dynamically with innerHTML
  • How to choose the right output method for debugging, UI, or DOM manipulation

📄 Common Output Methods

  • document.write() — writes directly to the page (not recommended for modern use)
  • console.log() — logs messages to the browser console (great for debugging)
  • alert() — shows a popup message
  • innerHTML — updates content of an HTML element
