üß† Notes

  • true and false are lowercase and unquoted
  • Falsy values: 0, -0, "", null, undefined, NaN, false
  • Truthy values: any non-zero number, non-empty string, object, array
  • Boolean() converts values to true/false
  • new Boolean() creates an object ‚Äî avoid it
  • Comparing Boolean objects can produce unexpected results
