JavaScript Object Construction

JavaScript objects can be created using different techniques. Each method offers flexibility depending on the structure, inheritance, and reuse you need.

üìÑ Construction Methods

// Object literal
let user1 = {
  name: "User",
  age: 25
};

// Constructor function
function User(name, age) {
  this.name = name;
  this.age = age;
}
let user2 = new User("Alex", 30);

// Object.create()
let baseUser = {
  greet() {
    return "Hello, " + this.name;
  }
};
let user3 = Object.create(baseUser);
user3.name = "Sam";

Use literals for quick objects, constructors for reusable templates, and Object.create() for prototypal inheritance.
