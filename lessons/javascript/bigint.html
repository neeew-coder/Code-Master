JavaScript BigInt

BigInt is a special numeric type for integers larger than Number.MAX_SAFE_INTEGER. It allows precise arithmetic with very large values.

📄 Declaring BigInts

let regular = 9007199254740991;       // Max safe integer
let big = 9007199254740993n;          // BigInt literal
let fromString = BigInt("123456789012345678901234567890");

console.log(typeof big);              // "bigint"

Add n to the end of a number or use BigInt() to create one. BigInts are a separate type and cannot mix directly with regular numbers.

🧠 Notes

  • BigInt handles integers beyond 2⁵³ - 1
  • Use n suffix or BigInt() constructor
  • Cannot mix BigInt with Number in operations
  • Use typeof to check: returns "bigint"
  • Useful for cryptography, finance, and precise counters
