JavaScript Array Iteration

Iteration methods allow you to loop through arrays and apply logic to each item. Some return new arrays, others return single values or booleans.

🔁 .map()

let nums = [1, 2, 3, 4, 5];
nums.map(x => x * 2); // [2, 4, 6, 8, 10]

🧠 Notes

  • .forEach() runs a function for each item (no return)
  • .map() returns a new array with transformed items
  • .filter() returns a new array with matching items
  • .reduce() combines items into a single value
  • .some() returns true if any match
  • .every() returns true if all match
