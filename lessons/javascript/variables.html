JavaScript Variables

Variables store data that your program can use and manipulate. In JavaScript, you can declare variables using let, const, or var.

📄 Variable Declaration

let name = "User";
const age = 20;
var city = "Sample City";

Use let for variables that may change, const for constants, and var (older syntax) for legacy code.

🧠 Notes

  • Variable names are case-sensitive and must start with a letter, _, or $
  • let and const are block-scoped (preferred)
  • var is function-scoped and can lead to unexpected behavior
  • Use const when the value should never change
