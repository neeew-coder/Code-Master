class Car {
  constructor(name, year) {
    this.name = name;
    this.year = year;
  }
  age() {
    const date = new Date();
    return date.getFullYear() - this.year;
  }
}
const myCar = new Car("Ford", 2014);
myCar.age(); // returns car age

ðŸ§  Lesson Summary

JavaScript classes provide a cleaner syntax for creating objects and handling inheritance. A class defines a blueprint with a constructor() and methods. You can create instances using new, and extend other classes with extends. The super() keyword allows child classes to access parent constructors. Classes are syntactic sugar over prototypes, making object-oriented patterns easier to read and maintain.
