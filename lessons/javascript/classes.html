ğŸ“… Class Method

class Car {
  constructor(name, year) {
    this.name = name;
    this.year = year;
  }
  age() {
    const date = new Date();
    return date.getFullYear() - this.year;
  }
}
const myCar = new Car("Ford", 2014);
myCar.age(); // returns car age
Output will appear here.

class Car {
  constructor(name, year) {
    this.name = name;
    this.year = year;
  }
  age(currentYear) {
    return currentYear - this.year;
  }
}
const myCar = new Car("Ford", 2014);
myCar.age(2025); // returns 11
