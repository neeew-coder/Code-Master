JavaScript DOM Events

DOM events let you respond to user interactions like clicks, typing, hovering, and more. You can attach event listeners to elements and define custom behavior when events occur.

let button = document.getElementById("myBtn");

button.addEventListener("click", function() {
  alert("Button clicked!");
});

Use addEventListener() to attach events. You can listen for click, input, mouseover, keydown, and more.

  • addEventListener() attaches multiple listeners to the same element

  • Inline onclick="..." is possible but less flexible
  • Events can be removed with removeEventListener()
