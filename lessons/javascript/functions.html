JavaScript Functions

Functions are reusable blocks of code that perform specific tasks. They help organize logic, reduce repetition, and make your code more readable and maintainable.

📄 Declaring a Function

function greet(name) {
  return "Hello, " + name + "!";
}

let message = greet("User"); // "Hello, User!"

Functions can accept parameters and return values. You call a function by using its name followed by parentheses.

📦 Anonymous & Arrow Functions

let greet = function(name) {
  return "Hi, " + name;
};

let shout = (text) => text.toUpperCase();

Anonymous functions are stored in variables. Arrow functions offer a shorter syntax and are common in modern JavaScript.
