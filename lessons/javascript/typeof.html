JavaScript typeof

The typeof operator returns the type of a variable or expression. It works on primitives and objects, but has quirks with null and arrays.

🔍 typeof on Primitives

typeof "John"         // "string"
typeof 3.14           // "number"
typeof true           // "boolean"
typeof 1234n          // "bigint"
typeof Symbol()       // "symbol"
typeof undefined      // "undefined"
typeof null           // "object"

📦 typeof on Objects

typeof new Map()       // "object"
