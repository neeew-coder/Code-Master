JavaScript Array Access

Arrays are indexed collections. You can access items using bracket notation, the .at() method, or format them using .join() and .toString().

📄 Access Methods

let fruits = ["Banana", "Orange", "Apple", "Mango"];

fruits[0];         // "Banana"
fruits.at(-1);     // "Mango" (ES2022)
fruits.length;     // 4
fruits.join(" - "); // "Banana - Orange - Apple - Mango"
fruits.toString(); // "Banana,Orange,Apple,Mango"

Use [index] or .at() to access items. Use .join() or .toString() to format arrays as strings.

🧠 Notes

  • [index] accesses items by position
  • .at() supports negative indexing (ES2022)
  • .length returns the number of items
  • .join() joins items with a custom separator
  • .toString() converts array to comma-separated string
