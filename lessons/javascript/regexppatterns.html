JavaScript RegExp Patterns

RegExp patterns define how strings are matched. Flags, character classes, metacharacters, assertions, and quantifiers shape the behavior of these matches.

🚩 Flag: /g (Global)

const text = "Is this all there is?";
const result = text.match(/is/g);
Output will appear here.

🔠 Character Class: [A-Z]

const text = "CodeMaster";
const result = text.match(/[A-Z]/g);
Output will appear here.

🔢 Metacharacter: \d

const text = "Give 100% effort!";
const result = text.match(/\d/g);
Output will appear here.

📍 Assertion: ^ (Start of Line)

const text = "CodeMaster rocks!";
const result = text.match(/^CodeMaster/);
Output will appear here.

➕ Quantifier: o+

const text = "Hellooo CodeMaster!";
const result = text.match(/o+/g);
Output will appear here.
