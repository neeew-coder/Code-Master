JavaScript Math.random()

Math.random() returns a floating-point number between 0 (inclusive) and 1 (exclusive). You can combine it with Math.floor() to generate random integers.

📄 Random Float

Math.random(); // returns a float between 0 and 1

📄 Random Integers

Math.floor(Math.random() * 10);       // 0–9
Math.floor(Math.random() * 100) + 1;  // 1–100

📄 Custom Functions

function getRndInteger(min, max) {
  return Math.floor(Math.random() * (max - min)) + min;
}

function getRndIntegerInclusive(min, max) {
  return Math.floor(Math.random() * (max - min + 1)) + min;
}

🧠 Notes

  • Math.random() returns a float between 0 and 1
  • Use Math.floor() to convert to integers
  • Multiply by range size to scale output
  • Use + min to shift the range
  • Custom functions help reuse logic for inclusive/exclusive bounds
