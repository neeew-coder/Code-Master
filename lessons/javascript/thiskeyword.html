JavaScript this Keyword

The this keyword refers to the object that is executing the current function. Its value depends on how the function is called.

👤 this in Object Method

const person = {
  firstName: "John",
  lastName: "Doe",
  fullName: function() {
    return this.firstName + " " + this.lastName;
  }
};
person.fullName(); // "John Doe"
Output will appear here.

🌍 this in Global Scope

let x = this; // refers to window in browsers
Output will appear here.

🔒 this in Strict Mode

"use strict";
function myFunction() {
  return this; // undefined
}
Output will appear here.

📞 Explicit Binding with call()

const person1 = {
  fullName: function() {
    return this.firstName + " " + this.lastName;
  }
};
const person2 = {
  firstName: "John",
  lastName: "Doe"
};
person1.fullName.call(person2); // "John Doe"
Output will appear here.
