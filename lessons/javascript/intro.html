JavaScript Introduction

JavaScript is a scripting language that runs in the browser — enabling interactivity, logic, and dynamic content. It powers everything from button clicks to animations, form validation, and full web apps.

📄 How JavaScript Works

  • Runs inside the browser’s JavaScript engine (like V8 in Chrome)
  • Executes code line-by-line, top to bottom
  • Can manipulate HTML and CSS via the DOM
  • Responds to user actions: clicks, typing, scrolling

🧩 Basic Syntax

console.log("Hello, JavaScript!");

console.log() outputs messages to the browser’s developer console — useful for testing and debugging.

🧠 Notes

  • JavaScript is case-sensitive: console ≠ Console
  • Semicolons are optional but recommended for clarity
  • Use console.log() to test values and flow
  • JavaScript runs after the HTML loads — place scripts at the bottom or use defer
