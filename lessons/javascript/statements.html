JavaScript Statements

A JavaScript statement is a command that tells the browser to do something. Statements are the building blocks of JavaScript programs — they define actions, logic, and flow.

📄 Example Statements

let x = 5;
let y = 10;
let sum = x + y;
console.log(sum);

Each line above is a statement. Together, they declare variables, perform a calculation, and output the result.

🧠 Notes

  • Statements end with a semicolon (;) — optional but recommended
  • JavaScript executes statements top to bottom, line by line
  • Blocks of statements are grouped using curly braces { }
  • Statements can include variable declarations, function calls, loops, and conditions
