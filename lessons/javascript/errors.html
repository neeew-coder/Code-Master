JavaScript Errors

JavaScript uses try, catch, finally, and throw to handle errors and control program flow. Errors are represented by the built-in Error object.

🧪 try...catch

try {
  adddlert("Welcome guest!");
} catch (err) {
  output = err.message;
}

🚨 throw Custom Error

try {
  throw "Too big";
} catch (err) {
  output = "Error: " + err;
}
Output will appear here.

🔁 finally Block

try {
  throw "fail";
} catch (err) {
  output = "Caught: " + err;
} finally {
  output += " | Finally executed";
}
Output will appear here.
