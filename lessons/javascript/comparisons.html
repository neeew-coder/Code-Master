üß† Notes

  • == compares value only; === compares value and type
  • != and !== are not equal and not equal type
  • && requires both conditions true; || requires one
  • ! negates a condition
  • ?? returns first non-nullish value
  • ?. safely accesses properties of possibly null/undefined objects
  • Use Number() to convert strings before comparing numerically
