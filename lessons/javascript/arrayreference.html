JavaScript Array Reference

Arrays are objects in JavaScript. When you assign an array to another variable, both variables point to the same reference. Changes affect both.

console.log(b); // ["x", "y", "z"]

üß† Notes

  • Arrays are objects and copied by reference
  • Mutating one reference affects all linked variables
  • Use [...array] or slice() to copy safely
  • a === b checks if two variables share the same reference
  • Shallow copies only duplicate top-level values
