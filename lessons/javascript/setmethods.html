🧠 Notes

  • add() inserts values — duplicates are ignored
  • size returns the number of elements
  • has() checks for membership
  • forEach() loops through all elements
  • values(), keys(), and entries() return iterators
  • keys() and values() are identical in Sets
  • entries() returns [value, value] pairs for compatibility with Maps
