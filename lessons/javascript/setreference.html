Method Description

difference()Returns the difference between two sets
entries()Returns [value,value] pairs
forEach()Invokes a callback for each element
has()Returns true if a value exists
intersection()Returns common elements
isDisjointFrom()Returns true if no shared elements
isSubsetOf()Returns true if subset
isSupersetOf()Returns true if superset
keys()Same as values()
symmetricDifference()Returns elements in A or B but not both
union()Returns combined elements
values()Returns all values
