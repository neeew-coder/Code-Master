JavaScript String Methods

JavaScript strings come with built-in methods that help you transform, search, and format text. These methods return new strings and never modify the original.

📘 What You’ll Learn

  • How to extract parts of strings using slice() and substring()
  • How to replace content using replace()
  • How to change case with toUpperCase() and toLowerCase()
  • How to clean whitespace using trim()
  • How to split strings into arrays using split()

📄 Common Methods

let text = "  Hello World  ";

text.toLowerCase();   
text.trim();           
text.startsWith("  H");
text.split(" ");      
text.concat("!!!");

These methods are useful for cleaning input, formatting output, and parsing strings into arrays.

🧠 Notes

  • .toLowerCase() and .toUpperCase() change case
  • .trim() removes leading/trailing spaces
  • .startsWith() checks how a string begins
  • .split() turns a string into an array
  • .concat() joins strings together
