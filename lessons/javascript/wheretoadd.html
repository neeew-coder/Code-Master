Where to Place JavaScript

JavaScript can be placed in different parts of your HTML file — each location affects when and how the code runs. You can write scripts inline, in the <head>, at the end of <body>, or in external files.

📄 Inline Script

<script>
  console.log("Inline script runs immediately");
</script>

Placed directly in the HTML — runs as soon as the browser reaches it. If placed in the <head>, it may run before the DOM is ready.

📦 External Script

<script src="script.js"></script>

Loads JavaScript from a separate file — keeps HTML clean and makes code reusable. Best placed at the end of <body> or with defer.

⏱️ Script Placement Options

Location Behavior Best Use
<head> Runs before DOM loads Use with defer or async
End of <body> Runs after HTML is loaded Safest for DOM manipulation
External file Loaded separately Keeps code modular
