JavaScript Objects

Objects are collections of key-value pairs. They’re used to group related data and behavior together. In JavaScript, almost everything is an object — including arrays and functions.

📄 Creating an Object

let user = {
  name: "User",
  age: 25,
  isStudent: true,
  greet: function() {
    return "Hello, " + this.name;
  }
};

console.log(user.name);       // "User"
console.log(user["age"]);     // 25
console.log(user.greet());    // "Hello, User"

Use dot notation (user.name) or bracket notation (user["name"]) to access properties. Functions inside objects are called methods.

🧠 Notes

  • Objects store data as key-value pairs
  • Keys are always strings (even if you don’t quote them)
  • Use this inside methods to refer to the object itself
  • Objects can contain other objects, arrays, or functions
  • Bracket notation is useful when keys are dynamic or contain spaces
