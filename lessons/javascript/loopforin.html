JavaScript For In Loop

The for...in loop iterates over the enumerable properties of an object. It can also be used on arrays, but is discouraged when index order matters.

üß† Notes

  • for...in loops over keys in objects
  • Use object[key] to access values
  • for...in can loop over arrays, but index order is not guaranteed
  • forEach() is preferred for ordered array iteration
  • forEach() accepts (value, index, array) as arguments
  • for...of is better for value-based iteration (next module)
