JavaScript While & Do While Loops

Loops execute a block of code repeatedly while a condition is true. The while loop checks the condition first. The do...while loop runs once before checking.

üîç Comparing For vs While

// Using for loop
const cars = ["BMW", "Volvo", "Saab", "Ford"];
let i = 0;
let text = "";

for (; cars[i]; ) {
  text += cars[i] + " ";
  i++;
}

// Using while loop
i = 0;
text = "";

while (cars[i]) {
  text += cars[i] + " ";
  i++;
}

üß† Notes

  • while checks the condition before each iteration
  • do...while runs once before checking the condition
  • Always update your loop variable to avoid infinite loops
  • for loops are more compact when all expressions are known
  • while is useful when the condition depends on runtime logic
