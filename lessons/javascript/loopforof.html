JavaScript For Of Loop

The for...of loop iterates over the values of an iterable object. It works with arrays, strings, maps, sets, NodeLists, and more.

üìÑ Syntax

for (variable of iterable) {
  // code block to be executed
}

üìÑ Array Example

const cars = ["BMW", "Volvo", "Mini"];
let text = "";

for (let car of cars) {
  text += car + " ";
}

üß† Notes

  • for...of loops over values, not keys
  • Works with arrays, strings, maps, sets, NodeLists
  • Use const, let, or var to declare the loop variable
  • Introduced in ES6 (2015); not supported in Internet Explorer
  • Use for...in for object keys, for...of for iterable values
