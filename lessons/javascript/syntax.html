JavaScript Syntax

JavaScript syntax defines the set of rules for writing code. It includes how statements are structured, how variables are declared, and how functions and expressions are formed.

📄 Basic Syntax Example

let name = "User";
function greet(person) {
  return "Hello, " + person;
}
console.log(greet(name));

This example shows variable declaration, a function definition, and a console output — all using correct syntax.

🧠 Notes

  • Statements usually end with a semicolon (;)
  • JavaScript is case-sensitive: let ≠ Let
  • Curly braces { } group blocks of code
  • Functions must be declared before they’re called (unless using hoisting)
