JavaScript String Search

JavaScript provides several methods to search within strings. These help you check for substrings, find positions, and validate patterns.

let text = "JavaScript is powerful";

text.includes("Script");     // true
text.indexOf("power");       // 15
text.startsWith("Java");     // true
text.endsWith("ful");        // true

Use .includes() to check for presence, .indexOf() to find position, and .startsWith()/.endsWith() to validate boundaries.
