JavaScript Reserved Words

Reserved words are identifiers that cannot be used as variable names, function names, or labels. They are part of the JavaScript language syntax or built-in objects.

🚫 Naming Error Simulation

Invalid:
let class = "Math"; // ❌ SyntaxError

Valid:
let className = "Math"; // ✅
