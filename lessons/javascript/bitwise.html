JavaScript Bitwise Operations

Bitwise operators treat numbers as 32-bit integers and operate on their binary representations. They're useful for low-level tasks, flags, and performance tricks.

🔁 Decimal ↔ Binary Conversion

function dec2bin(dec) {
  return (dec >>> 0).toString(2);
}

function bin2dec(bin) {
  return parseInt(bin, 2).toString(10);
}
