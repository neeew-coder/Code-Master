JavaScript Assignment Operators

Assignment operators store values in variables. The most common is =, but JavaScript also supports compound operators like +=, *=, and %= for shorthand updates.

📄 Common Assignment Operators

  • = — Assigns a value
  • += — Adds and assigns
  • -= — Subtracts and assigns
  • *= — Multiplies and assigns
  • /= — Divides and assigns
  • %= — Modulus and assigns

📦 Syntax Example

let score = 10;

score += 5;   // score = score + 5 → 15
score *= 2;   // score = score * 2 → 30
score %= 7;   // score = score % 7 → 2

Compound assignment operators simplify repetitive updates. They’re especially useful in loops, counters, and calculations.

🧠 Notes

  • Assignment happens from right to left: x = y means x gets y’s value
  • Compound operators reduce repetition and improve readability
  • Use += for counters, *= for scaling, %= for wrapping values
  • Assignment operators work with numbers, strings, and even booleans
