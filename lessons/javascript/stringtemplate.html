JavaScript String Templates

Template literals use backticks (` `) to create flexible strings. You can embed variables, expressions, and even write multiline text without escape characters.

📄 Syntax Example

let name = "User";
let age = 25;

let message = \`Hello, \${name}. You are \${age} years old.\`;

let multiline = \`
Line 1
Line 2
Line 3
\`;

Use \${} to embed variables or expressions. Template literals improve readability and reduce string concatenation.

🧠 Notes

  • Use backticks ` ` instead of quotes
  • Embed variables with \${variable}
  • Supports multiline strings without \n
  • Great for dynamic messages and HTML templates
  • Template literals are part of ES6 (modern JavaScript)
