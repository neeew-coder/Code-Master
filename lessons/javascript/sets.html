JavaScript Sets

A JavaScript Set is a collection of unique values. Each value can only occur once. Values can be of any type — primitives or objects.

📦 Create from Array

const letters = new Set(["a", "b", "c"]);

🧠 Notes

  • Use add() to insert values

  • instanceof Set returns true for Set instances
