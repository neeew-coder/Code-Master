JavaScript Date Formats

JavaScript supports multiple date input formats. The most reliable is ISO 8601. Other formats may vary across browsers and time zones.

new Date("2024-03-25T12:00:00Z"); // UTC time

üß† Notes

  • ISO format (YYYY-MM-DD) is safest and most consistent
  • Short and long formats may vary across browsers
  • Month names are case-insensitive and commas are ignored
  • Date.parse() returns milliseconds since Jan 1, 1970
  • Time zones affect interpretation of partial dates
