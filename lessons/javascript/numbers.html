JavaScript Numbers

Numbers in JavaScript include integers, decimals, and special values like NaN and Infinity. You can perform math operations and use built-in methods to format or validate them.

📄 Declaring Numbers

let x = 42;
let y = 3.14;
let z = "100";

Number(z);         // 100
parseInt("50px");  // 50
parseFloat("2.5em"); // 2.5

Use Number(), parseInt(), and parseFloat() to convert strings to numbers. JavaScript treats all numbers as floating-point under the hood.

📦 Common Methods

let value = 5.6789;

value.toFixed(2);         // "5.68"
Math.round(value);        // 6
Math.floor(value);        // 5
Math.ceil(value);         // 6
Number.isNaN("hello");    // false
Number.isFinite(100 / 0); // false

Use these methods to round, format, and validate numbers. Math is a built-in object with many useful utilities.

🧠 Notes

  • JavaScript uses 64-bit floating-point numbers
  • NaN means “Not a Number” — often from invalid math
  • Infinity results from division by zero
  • Math provides rounding, random numbers, and more
  • Use Number() to convert strings safely
