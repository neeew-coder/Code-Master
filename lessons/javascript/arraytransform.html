JavaScript Array Transformation

Transformation methods return modified versions of arrays. Some mutate the original array, while others return a new one. Use them to merge, extract, or restructure data.

🧠 Notes

  • .concat() merges arrays without modifying originals
  • .slice() extracts a portion into a new array
  • .splice() modifies the original array
  • .toSpliced() (ES2023) returns a modified copy
  • Use .slice() and .toSpliced() for safer operations
