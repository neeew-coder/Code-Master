JavaScript Comments

Comments are notes in your code that the browser ignores. They help explain logic, disable code temporarily, and make your scripts easier to read and maintain.

  • How to use comments for debugging and code clarity

📄 Comment Syntax

// This is a single-line comment

/*
  This is a multi-line comment
  spanning multiple lines
*/

let x = 5; // Inline comment

Use // for single-line comments and /* */ for multi-line blocks. Comments do not affect code execution.

🧠 Notes

  • Disable code temporarily by commenting it out
  • Comments improve collaboration and debugging
  • Never leave sensitive info in comments — they’re visible in source
