JavaScript Break and Continue

The break statement "jumps out" of a loop or block. The continue statement "skips" one iteration and continues with the next.

üß† Notes

  • break exits a loop or block immediately
  • continue skips the current iteration
  • Labels allow break or continue to target specific blocks
  • break without label works in loops and switch
  • continue only works inside loops
