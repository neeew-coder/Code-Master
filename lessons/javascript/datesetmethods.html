JavaScript Set Date Methods

Set methods allow you to change parts of a date object — year, month, day, hour, minute, second, or millisecond. Overflow values automatically adjust the date.

📅 setFullYear()

const d = new Date("January 01, 2025");
d.setFullYear(2020);            // Year only
d.setFullYear(2020, 11, 3);     // Year, month, day

Output will appear here.

📆 setDate()

d.setDate(15);               // 15th day
d.setDate(d.getDate() + 50); // Adds 50 days
▶️ Run setDate
Output will appear here.

⏰ setHours()

d.setHours(22);              // 10 PM
d.setHours(22, 10, 20);      // 10:10:20 PM
▶️ Run setHours
Output will appear here.

🕒 setMinutes(), setSeconds(), setMilliseconds()

d.setMinutes(30);        // 30 minutes
d.setSeconds(45);        // 45 seconds
d.setMilliseconds(500);  // Half a second

📍 setTime()

d.setTime(86400000); // Jan 2, 1970
▶️ Run setTime
Output will appear here.

🧠 Notes

  • setDate() accepts 1–31 and handles overflow
  • setFullYear() can also set month and day
  • setHours() can also set minutes and seconds
  • setTime() sets milliseconds since Jan 1, 1970
  • Use getDate() + N to add days
