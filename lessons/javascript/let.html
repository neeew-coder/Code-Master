JavaScript let

The let keyword declares a block-scoped variable. It’s the modern way to define variables that can change, and it avoids many of the pitfalls of var.

📄 Syntax Example

let count = 1;
count = count + 1;
console.log(count); // Outputs: 2

You can reassign a let variable, but you cannot redeclare it in the same block.

🧠 Notes

  • let is block-scoped — it only exists within { }
  • You can reassign a let variable, but not redeclare it in the same scope
  • Use let when the value will change over time
  • Prefer let over var for modern JavaScript
