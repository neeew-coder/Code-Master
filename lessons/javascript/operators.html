JavaScript Operators

Operators are symbols that perform operations on variables and values. JavaScript includes arithmetic, assignment, comparison, logical, and more — each with specific behavior and precedence.

📄 Common Operator Types

  • Arithmetic: +, -, *, /, %
  • Assignment: =, +=, *=, etc.
  • Comparison: ==, ===, !=, >, <
  • Logical: &&, ||, !
  • String Concatenation: + used with strings

📦 Syntax Example

let a = 10;
let b = 5;

let sum = a + b;       // 15
let diff = a - b;      // 5
let isEqual = a > b; // true
let result = (a > b) && (b > 0); // true

Operators can be combined to form expressions. Use parentheses ( ) to control precedence and clarity.

🧠 Notes

  • === checks both value and type — prefer it over ==
  • Logical operators return boolean values: true or false
  • Use + for both math and string joining — context matters
  • Operator precedence affects how expressions are evaluated — use parentheses to clarify
