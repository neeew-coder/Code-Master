JavaScript Typed Arrays

Typed arrays are fixed-length, binary-safe views into raw memory buffers. They’re ideal for performance-critical tasks like graphics, media, and file manipulation.

📘 Typed Array Types

Name Range Bytes Type
Int8Array-128 to 1271byte
Uint8Array0 to 2551octet
Uint8ClampedArray0 to 2551octet (clamped)
Int16Array-32,768 to 32,7672short
Uint16Array0 to 65,5352unsigned short
Int32Array-2^31 to 2^31-14long
Uint32Array0 to 2^32-14unsigned long
Float32Array±3.4e384float
Float64Array±1.8e3088double

🔢 Int8Array

▶️ Create Int8Array 
const myArr = new Int8Array(10);
myArr[0] = 127;
myArr[1] = -128;
myArr[2] = 200; // wraps to -56

🚫 Uint8ClampedArray

▶️ Clamp Behavior 
const myArr = new Uint8ClampedArray(3);
myArr[0] = 100;
myArr[1] = 300; // clamped to 255
myArr[2] = -50; // clamped to 0
