JavaScript Modules (Simplified)

JavaScript modules let you split code into reusable files using export and import. This makes your code easier to maintain, test, and organize.

  • Named exports let you export multiple values by name
  • Default exports let you export a single main value
  • Modules must use type="module" in the script tag
  • Modules only work over HTTP(s) ‚Äî not file://

JavaScript Modules (Simplified)

JavaScript modules let you split code into reusable files using export and import. This makes your code easier to maintain, test, and organize.

  • Named exports let you export multiple values by name
  • Default exports let you export a single main value
  • Modules must use type="module" in the script tag
  • Modules only work over HTTP(s) ‚Äî not file://

üìö Syntax Reference

Named Export:

Default Export:

export default message;

Import:

import { name, age } from "./person.js";
import message from "./message.js";
