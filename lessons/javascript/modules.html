ðŸ“š Syntax Reference

Named Export:

export const name = "Jesse";
export const age = 40;

Default Export:

const message = () => `${name} is ${age} years old.`;
export default message;

Import:

import { name, age } from "./person.js";
import message from "./message.js";

ðŸ§  Lesson Summary

JavaScript modules allow you to split your code into separate files and share functionality using export and import. Named exports let you export multiple items, while default exports simplify single-entry modules. Modules help avoid global namespace conflicts, improve maintainability, and support scalable architecture. To use modules in the browser, scripts must be marked with type="module", and bundlers can optimize them for production.
