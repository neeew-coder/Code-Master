JavaScript Strings

Strings are sequences of characters used to represent text. JavaScript provides many methods to inspect, modify, and format strings.

📄 Declaring Strings

let name = "User";
let greeting = 'Hello';
let phrase = \`Welcome, \${name}!\`; // Template literal

Strings can be declared using double quotes, single quotes, or backticks for template literals. Use \${} inside backticks to embed expressions.

📦 Common Methods

let text = "JavaScript";

text.length;           // 10
text.toUpperCase();    // "JAVASCRIPT"
text.slice(4);         // "Script"
text.includes("Script"); // true
text.replace("Java", "Type"); // "TypeScript"

Use string methods to transform, search, and extract content. Strings are immutable — methods return new strings.
