JavaScript const

The const keyword declares a block-scoped constant. Once assigned, its value cannot be changed. It’s ideal for values that should remain fixed throughout your program.

📄 Syntax Example

const pi = 3.14159;
console.log(pi); // Outputs: 3.14159

pi = 3.14; // ❌ Error: Assignment to constant variable

You can’t reassign a const variable. However, if it holds an object or array, you can still modify its contents.

🧠 Notes

  • const must be initialized when declared
  • You cannot reassign a const variable
  • Objects and arrays declared with const can still be modified
  • Use const for values that should never change
