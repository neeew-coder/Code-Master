JavaScript Object Properties

Object properties are the named values inside an object. You can access, update, add, or delete properties using dot or bracket notation. Properties can hold any data type ‚Äî including functions, arrays, or other objects.

üìÑ Property Access & Modification

let user = {
  name: "User",
  age: 25
};

// Access
console.log(user.name);        // "User"
console.log(user["age"]);      // 25

// Update
user.age = 26;

// Add
user.isStudent = true;

// Delete
delete user.name;

Use dot notation for known keys and bracket notation for dynamic or special keys. Deleting a property removes it from the object entirely.
