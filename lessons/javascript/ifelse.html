JavaScript If Else Statements

Conditional statements control the flow of your program. Use if, else if, and else to execute code based on Boolean logic.

let hour = 14;

if (hour < 12) {
  greeting = "Good morning";
} else if (hour < 18) {
  greeting = "Good afternoon";
} else {
  greeting = "Good evening";
}

const hours = [8, 14, 20];

  if (hour < 12) {
    console.log("Good morning");
  } else if (hour < 18) {
    console.log("Good afternoon");
  } else {
    console.log("Good evening");
  }

üß† Notes

  • if runs when the condition is true
  • else if checks additional conditions
  • else runs when no previous condition matched
  • Conditions must evaluate to Boolean values
  • Use === for strict comparisons
  • Use Number() to convert input before comparing
