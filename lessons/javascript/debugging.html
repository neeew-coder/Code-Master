JavaScript Debugging

Debugging is the process of finding and fixing errors in your code. JavaScript provides tools like console.log(), debugger, and browser developer tools to help you track down issues.

  • Use console.log() to print values to the browser console
  • Use debugger to pause execution and inspect variables
  • Open your browser’s developer tools with F12 or right-click → Inspect
  • Set breakpoints to pause code and step through execution

🧪 Debugging Preview

let a = 5;
let b = 6;
let c = a + b;
console.log(c); // 11
debugger;
document.getElementById("output").innerText = c;
Output will appear here.

📚 Debugging Tools Reference

console.log() — Print values to the console

console.log("Debug message");

debugger — Pause execution in dev tools

let x = 15 * 5;
debugger;
console.log(x);

Browser Console Access:

  • Chrome: F12 → Console
  • Firefox: F12 → Web Console
  • Edge: F12 → Console
  • Safari: Preferences → Advanced → Enable Develop menu → Show Error Console
