JavaScript Iterator Helpers

ECMAScript 2025 introduces powerful helper methods for iterators, enabling functional transformations like map(), filter(), reduce(), and more — directly on iterator streams.

📥 Iterator.from()

▶️ Create Iterator 
const myIterator = Iterator.from([1, 2, 3]);
for (const x of myIterator) {
  console.log(x);
}
Output will appear here.

🧠 map()

▶️ Transform Elements 
const myIterator = Iterator.from("123456");
const mapped = myIterator.map(x => x * 2);
Output will appear here.

🔍 filter()

▶️ Filter Elements 
const myIterator = Iterator.from([32, 33, 16, 40]);
const filtered = myIterator.filter(x => x > 18);
Output will appear here.

🧩 flatMap()

▶️ Expand Elements 
const myIterator = Iterator.from([1, 2, 3]);
const expanded = myIterator.flatMap(x => [x, x * 10]);
Output will appear here.

🎯 take()

▶️ Take Elements 
const myIterator = Iterator.from([1, 2, 3, 4, 5]);
const firstThree = myIterator.take(3);
Output will appear here.

🚫 drop()

▶️ Drop Elements 
const myIterator = Iterator.from([1, 2, 3, 4, 5]);
const afterTwo = myIterator.drop(2);
