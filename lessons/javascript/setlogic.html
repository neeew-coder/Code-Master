JavaScript Set Logic

JavaScript 2025 introduced new logical methods for Set objects: union(), intersection(), difference(), symmetricDifference(), isSubsetOf(), isSupersetOf(), and isDisjointFrom().

  function runSymmetricDifference() {
    const A = new Set(['a', 'b', 'c']);
    const B = new Set(['b', 'c', 'd']);
    const C = new Set([...A, ...B].filter(x => !(A.has(x) && B.has(x))));
    document.getElementById("symmetricOutput").innerHTML = [...C].join(", ");
  }

üß† Notes

  • union() combines all elements from both sets
  • intersection() returns common elements
  • difference() returns elements in A but not in B
  • symmetricDifference() returns elements in A or B but not both
  • isSubsetOf() checks if all elements in A are in B
  • isSupersetOf() checks if all elements in B are in A
  • isDisjointFrom() returns true if A and B share no elements
  • All methods return new sets or booleans ‚Äî original sets remain unchanged
  • These methods are supported in modern browsers since mid-2024
