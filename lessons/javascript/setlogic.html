  • union() combines all elements from both sets
  • intersection() returns common elements
  • difference() returns elements in A but not in B
  • symmetricDifference() returns elements in A or B but not both
  • isSubsetOf() checks if all elements in A are in B
  • isSupersetOf() checks if all elements in B are in A
  • isDisjointFrom() returns true if A and B share no elements
  • All methods return new sets or booleans â€” original sets remain unchanged
