JavaScript Number Methods

Number methods format and convert numeric values. These instance methods return strings and are useful for display, precision control, and base conversion.

🧠 Notes

  • .toFixed() returns a string with fixed decimals
  • .toExponential() formats in scientific notation
  • .toPrecision() controls total digits (not just decimals)
  • .toString() converts a number to text
  • .toString(radix) converts to binary, octal, or hex
