JavaScript Maps

A Map object holds key-value pairs where keys can be any datatype. Maps remember the original insertion order of keys and are directly iterable.

📦 Create from Array

const fruits = new Map([
  ["apples", 500],
  ["bananas", 300],
  ["oranges", 200]
]);
