JavaScript Regular Expressions

Regular expressions (RegExp) are patterns used to match character combinations in strings. They’re powerful tools for search, replace, and validation.

🔍 match()

const text = "Visit CodeMaster!";
const result = text.match(/codemaster/i);
Output will appear here.

♻️ replace()

const text = "Visit Microsoft!";
const result = text.replace(/Microsoft/i, "CodeMaster");
Output will appear here.

🔎 search()

const text = "Visit CodeMaster!";
const index = text.search(/CodeMaster/);
Output will appear here.

🔠 Character Class [CM]

const text = "CodeMaster";
const result = text.match(/[CM]/g);
Output will appear here.

➕ Quantifier o+

const text = "Hellooo CodeMaster!";
const result = text.match(/o+/g);
Output will appear here.
