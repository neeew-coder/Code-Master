HTML Computer Code Elements

HTML provides semantic tags for displaying programming logic, variables, user input, and code output. These elements are displayed using the browser's default monospace font for clarity.

HTML Code Tags Explained walks through semantic tags for displaying code snippets, keyboard input, sample output, and variables — with clean examples and no iframe detour.

📘 What You’ll Learn

  • How to display inline and block code using <code> and <pre>
  • How to represent keyboard input using <kbd>
  • How to simulate program output with <samp>
  • How to mark variables in logic, math, or programming using <var>
  • How semantic tags improve accessibility and clarity in technical documentation

🧑‍💻 <code> — Display Computer Code

Use <code> to mark up short snippets of code or inline logic.

x = 5;
y = 6;
z = x + y; 
<code>
x = 5;
y = 6;
z = x + y;
</code>

📐 <pre> — Preserve Formatting

Use <pre> to preserve spacing, line breaks, and tabs. It's often combined with <code> for readable code blocks. 

<pre>
  <code>
    x = 5;
    y = 6;
    z = x + y;
  </code>
</pre>

⌨️ <kbd> — Keyboard Input

Use <kbd> to show keys users should press — commonly used in documentation or tutorials.

Save by pressing Ctrl + S

<p>Save by pressing <kbd>Ctrl + S</kbd></p>

🖥️ <samp> — Sample Output

Use <samp> to display simulated output from a program.

Message from my computer:
File not found.
Press F1 to continue 
<p>Message from my computer:</p>
<p><samp>File not found.<br>Press F1 to continue</samp></p>

🧮 <var> — Variables in Logic

Use <var> to identify variables in code or math expressions. Most browsers render it in italics.

The area of a triangle is: 1/2 × b × h, where b is base, and h is height.

<p>
  The area of a triangle is: 1/2 × <var>b</var> × <var>h</var>, 
  where <var>b</var> is base, and <var>h</var> is height.
</p>

🧠 Lesson Summary

  • <code> is for short code snippets
  • <pre> preserves line breaks and spacing
  • <kbd> shows keys users should press
  • <samp> displays sample output
  • <var> identifies variables in expressions
