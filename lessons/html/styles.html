HTML Styles

In this tutorial, you'll learn how to use the style attribute to apply inline CSS styles to HTML elements — including text color, background, fonts, size, and alignment.

📘 What You’ll Learn

  • How to use style="" to apply CSS directly to an element
  • How to format text, background, borders, and spacing inline
  • How inline styles override external and internal CSS
  • Why inline styles should be limited to quick edits or dynamic overrides

🎨 Syntax

Use the style attribute like this: 

<tagname style="property:value;">

🖌️ Text Color

Use the color property to change text color. 

<h1 style="color:blue;">This is a heading</h1>
<p style="color:red;">This is a paragraph.</p>

🧱 Background Color

Use background-color to set background colors. 

<h1 style="background-color:powderblue;">This is a heading</h1>
<p style="background-color:tomato;">This is a paragraph.</p>

🔤 Fonts

Use font-family to change the font style. 

<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>

🔠 Text Size

Use font-size to control text size. 

<h1 style="font-size:300%;">This is a heading</h1>
<p style="font-size:160%;">This is a paragraph.</p>

📐 Text Alignment

Use text-align to align text horizontally. 

<h1 style="text-align:center;">Centered Heading</h1>
<p style="text-align:center;">Centered paragraph.</p>

🧠 Lesson Summary

  • style="color: orange;" changes text color
  • style="background-color: #f0f0f0;" sets background
  • style="margin: 1rem;" adds spacing
  • Inline styles have high specificity and override most external rules
