HTML <input> Attributes

Input attributes extend the behavior of form fields. They control default values, formatting rules, accessibility, and validation. This module explores commonly used attributes with real examples.

🔡 Common Display Attributes

  • value — Sets initial content
  • readonly — Prevents edits (still submits value)
  • disabled — Prevents edits (excluded from submission)
  • size — Visible width in characters (default is 20)
  • maxlength — Limits number of allowed characters
<input type="text" value="John" readonly size="30" maxlength="50">

📏 Validation & Format Control

  • min / max — Allowed range
  • step — Legal intervals (e.g. step="3" gives 3, 6, 9...)
  • pattern — Regular expression to validate input
  • required — Must be filled before form submit
<input type="number" min="1" max="5">
<input type="text" pattern="[A-Za-z]{3}" title="Three letters only" required>

💡 Accessibility & Usability

  • placeholder — Shows example or prompt text
  • autofocus — Focuses field when page loads
  • autocomplete — Allows browser to suggest values
  • list — Connects to <datalist> with options
<input type="text" placeholder="123-45-678" pattern="[0-9]{3}-[0-9]{2}-[0-9]{3}">
<input type="text" list="browsers">
<datalist id="browsers">
  <option value="Edge">
  <option value="Firefox">
  <option value="Chrome">
</datalist>

🖼️ Input Image Attributes

  • width / height — Size input image area
<input type="image" src="img_submit.gif" alt="Submit" width="48" height="48">
