HTML Comments

HTML comments are not displayed in the browser, but they help document your code and organize your markup.

💬 Syntax

Use the following syntax to add a comment: 

<!-- Write your comments here -->

📝 Adding Comments

Use comments to leave notes or reminders in your code. 

<!-- This is a comment -->

<p>This is a paragraph.</p>

<!-- Remember to add more information here -->

This is a paragraph.

🙈 Hiding Content

Comments can be used to temporarily hide content from the browser. 

<p>This is a paragraph.</p>

<!-- <p>This is another paragraph</p> -->

<p>This is a paragraph too.</p>

This is a paragraph.

This is a paragraph too.

📦 Hiding Multiple Lines

Everything between <!-- and --> will be hidden. 

<p>This is a paragraph.</p>

<!--
<p>Look at this cool image:</p>
<img border="0" src="pic_trulli.jpg" alt="Trulli">
-->

<p>This is a paragraph too.</p>

This is a paragraph.

This is a paragraph too.

🔍 Hide Inline Content

You can hide parts of a line using comments. 

<p>This <!-- great text --> is a paragraph.</p>

This is a paragraph.
