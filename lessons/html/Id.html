HTML id Attribute

The id attribute assigns a unique identifier to an HTML element. It must be unique on the page and is commonly used for styling, scripting, and linking.

📌 Key Features

  • Uniquely identifies an element
  • Used with CSS using #idName
  • Accessed by JavaScript using getElementById()
  • Used to create bookmarks and navigation targets

🎨 Styling an ID with CSS

#myHeader {
  background-color: lightblue;
  color: black;
  padding: 40px;
  text-align: center;
}
...
<h1 id="myHeader">My Header</h1>

My Header

🔍 JavaScript Access

You can manipulate an element using JavaScript by its ID:

function updateHeader() {
  document.getElementById("myHeader").innerHTML = "Welcome to CodeMaster!";
}

🔗 Bookmark Navigation

Jump to sections using ID-based anchors: 

<h2 id="chapter4">Chapter 4</h2>
<a href="#chapter4">Jump to Chapter 4</a>
