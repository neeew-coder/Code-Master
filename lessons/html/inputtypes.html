HTML <input> Types

The <input> element supports a wide variety of type attributes to handle specific kinds of user input. Each type offers tailored behavior, validation, and user interface components.

🔤 Textual Input Types

<input type="text">            <!-- Default: single-line text -->
<input type="email">           <!-- Validates email format -->
<input type="password">        <!-- Obscures typed characters -->
<input type="search">          <!-- Similar to text, optimized for search -->
<input type="tel">             <!-- Phone number formatting (mobile friendly) -->
<input type="url">             <!-- Validates URL format -->

📌 Tip: The default value for type is text.

📅 Date & Time Inputs

<input type="date">             <!-- Date picker -->
<input type="time">             <!-- Select hour/minute -->
<input type="datetime-local">   <!-- Combined date + time picker -->
<input type="month">            <!-- Year-month selection -->
<input type="week">             <!-- Select a week in a year -->

🧠 These inputs often render native pickers on mobile and desktop browsers.

🔢 Numeric & Range Inputs

<input type="number">           <!-- Accepts numeric input with optional min/max -->
<input type="range">            <!-- Slider bar with numeric values -->

🛠 Add min, max, and step to fine-tune behavior.

☑️ Selection & File Inputs

<input type="checkbox">         <!-- Multiple choice selections -->
<input type="radio">            <!-- Single choice (name must match) -->
<input type="file">             <!-- Upload file from device -->
<input type="color">            <!-- Color picker -->
<input type="hidden">           <!-- Invisible input for static data -->

📂 File inputs support multiple files and accepted MIME types using attributes like multiple and accept.

🎯 Button & Action Inputs

<input type="submit">           <!-- Submits form data -->
<input type="reset">            <!-- Resets form fields -->
<input type="button" value="Click">  <!-- Generic clickable button -->
<input type="image" src="btn.png">  <!-- Use image as submit button -->

🎨 Prefer using <button> when you need inner HTML or icon/text combinations.

🧠 Lesson Summary

  • type="text" is the default input type
  • email, url, and tel offer built-in validation
  • date, time, and range provide native UI controls
  • checkbox and radio handle selection logic
  • file and color enhance interactivity
  • submit, reset, and button trigger form actions
