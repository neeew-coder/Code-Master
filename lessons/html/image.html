HTML Images

Images add visual appeal and communicate ideas effectively. In HTML, images are defined using the <img> element.

📘 What You’ll Learn

  • How to embed images using <img src="..." alt="...">
  • How to use src for file paths and alt for accessibility
  • How to resize images using width, height, and inline style
  • How to float images beside text using CSS
  • How to wrap images in clickable links using <a>
  • How to organize image folders and reference relative vs. absolute paths

📷 Basic Syntax

The <img> tag is empty — it holds no content, only attributes. Required attributes:

  • src – Image path or URL
  • alt – Text alternative if image fails
<img src="images/trulli.jpg" alt="Italian Trulli">

🔁 Broken Images & alt Text

If the image file is missing, the alt text will appear instead:

<img src="images/wrongname.jpg" alt="Trulli in Italy">

📏 Image Size

Use style, width, or height to resize:

<img src="images/girl.jpg" alt="Girl in a jacket" style="width:300px;height:400px;">
<img src="images/girl.jpg" alt="Girl in a jacket" width="300" height="400">

📁 Folder Paths

Include folder name in src if images are stored inside subfolders:

<img src="images/html5.png" alt="HTML5 Icon" style="width:128px;height:128px;">

🌀 Animated Images

GIFs allow simple animations without JavaScript:

<img src="images/loading.gif" alt="Loading icon" style="width:48px;height:48px;">

🔗 Image as a Link

Make an image clickable by wrapping it in a link tag:

<a href="lesson.html">
  <img src="images/smiley.png" alt="Lesson" style="width:42px;height:42px;">
</a>

🧭 Floating Images

Use float: left or float: right to position images beside text:

🧠 Lesson Summary

  • <img> embeds an image using src and alt
  • Use style, width, or height to resize
  • Broken images display the alt text
  • Use folder paths in src to organize assets
  • Wrap images in <a> to make them clickable
  • Use float to position images beside text
  • GIFs allow simple animations without JavaScript
