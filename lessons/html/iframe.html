HTML <iframe> Element

The <iframe> tag allows you to embed another webpage inside your current one — useful for tutorials, previews, external videos, or navigation panes.

📌 Syntax

<iframe src="URL" title="Description"></iframe>
  • src: URL of the embedded page
  • title: Provides a description for screen readers
---

📐 Set Height and Width

You can control the iframe size using HTML attributes or CSS:

<iframe src="https://codemaster.ph" height="300" width="500" title="CodeMaster Demo"></iframe>

<iframe src="https://codemaster.ph" style="height:300px; width:500px;" title="CodeMaster Demo"></iframe>
Iframe would load: https://codemaster.ph
---

🚫 Remove or Style Borders

Use inline CSS to control border visibility and style:

<iframe src="..." style="border:none;" title="No Border Iframe"></iframe>

<iframe src="..." style="border:2px solid red;" title="Red Border Iframe"></iframe>
---

🎯 Targeting an Iframe with a Link

Assign a name to your iframe and link using target: 

<iframe src="https://codemaster.ph" name="iframe_a" title="CodeMaster Target"></iframe>

🧠 Clicking the link loads Code Master inside the named iframe.---

📚 Summary

Feature Details
Tag <iframe>
URL attribute src
Accessibility title describes frame for screen readers
Sizing height & width or inline style
Border Control style="border:none;" or custom border
Navigation name on iframe + target on link
