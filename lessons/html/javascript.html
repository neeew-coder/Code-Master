Using JavaScript in HTML

JavaScript is a scripting language used to create dynamic, interactive behavior in websites. Common tasks include image manipulation, form validation, and content updates.

<script>
// JavaScript code goes here
</script>

You can embed JavaScript directly in the HTML document, or link an external script using:

<script src="script.js"></script>

üß† Selecting Elements with JavaScript

To target specific elements in HTML, use the document.getElementById() method. This allows you to modify content, styles, or attributes.

Change Content

document.getElementById("demo").innerHTML = "Hello JavaScript!";

Change Styles

document.getElementById("demo").style.fontSize = "25px";
document.getElementById("demo").style.color = "red";
document.getElementById("demo").style.backgroundColor = "yellow";

Change Attributes

document.getElementById("image").src = "picture.gif";
