HTML Paragraphs

In this tutorial, you'll learn how to use the <p> tag to create paragraphs, how browsers handle whitespace, and how to use line breaks and horizontal rules for better formatting.

📄 The <p> Element

The <p> tag defines a paragraph. It’s a block-level element that starts on a new line. 

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

🖥️ HTML Display Behavior

HTML ignores extra spaces and line breaks in source code. The browser collapses whitespace when rendering. 

<p>
This paragraph
contains a lot of lines
in the source code,
but the browser
ignores it.
</p>

<p>
This paragraph
contains         a lot of spaces
in the source         code,
but the        browser
ignores it.
</p>

📏 Horizontal Rules

The <hr> tag defines a thematic break and is often rendered as a horizontal line. 

<h1>This is heading 1</h1>
<p>This is some text.</p>
<hr>
<h2>This is heading 2</h2>
<p>This is some other text.</p>
<hr>

↩️ Line Breaks

Use <br> to insert a line break without starting a new paragraph. 

<p>This is<br>a paragraph<br>with line breaks.</p>

📜 The Poem Problem

HTML collapses whitespace, so poems written with line breaks will appear as a single line. 

<p>
  My Bonnie lies over the ocean.

  My Bonnie lies over the sea.

  My Bonnie lies over the ocean.

  Oh, bring back my Bonnie to me.
</p>

🧾 Solution: The <pre> Element

Use <pre> to preserve whitespace and line breaks. It displays text in a fixed-width font. 

<pre>
  My Bonnie lies over the ocean.

  My Bonnie lies over the sea.

  My Bonnie lies over the ocean.

  Oh, bring back my Bonnie to me.
</pre>

🧠 Lesson Summary

  • <p> defines a paragraph block
  • Browsers add default margin above and below each paragraph
  • Paragraphs cannot be nested — each must be standalone
  • Use multiple <p> tags to separate thoughts or sections
