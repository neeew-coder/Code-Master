HTML class Attribute

The class attribute lets you assign style rules to HTML elements. You can reuse class names across multiple elements, combine them, and target them with CSS or JavaScript.

üé® Styling with a Class

Define a class in CSS using a period (.) followed by the class name:

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>City Class Example</title>
  <style>
    .city {
      background-color: tomato;
      color: white;
      border: 2px solid black;
      margin: 20px;
      padding: 20px;
    }
  </style>
</head>
<body>

  <div class="city">
    <h2>London</h2>
    <p>London is the capital of England.</p>
  </div>

  <div class="city">
    <h2>Paris</h2>
    <p>Paris is the capital of France.</p>
  </div>

  <div class="city">
    <h2>Tokyo</h2>
    <p>Tokyo is the capital of Japan.</p>
  </div>

</body>
</html>

London

London is the capital of England.

Paris

Paris is the capital of France.

Tokyo

Tokyo is the capital of Japan.

üìö Multiple Classes

You can assign more than one class to an element. Just separate them by spaces: 

<h2 class="city main">London</h2>
<h2 class="city">Paris</h2>
<h2 class="city">Tokyo</h2>

The London heading will inherit styles from both .city and .main.

üîÅ Shared Classes Across Elements

Different element types can share the same class:

<h2 class="city">Paris</h2>
<p class="city">Paris is the capital of France</p>

üß† Using Class in JavaScript

You can target classes with JavaScript using getElementsByClassName(): 

function hideCities() {
  var x = document.getElementsByClassName("city");
  for (var i = 0; i < x.length; i++) {
    x[i].style.display = "none";
  }
}

Clicking a button could trigger this function to hide all elements with the class city.

üìå Syntax Reminder

  • Create a class in CSS using a period: .city { ... }
  • Add it to HTML using: class="city"
  • Class names are case sensitive
  • Multiple elements can share a class name
  • Multiple classes = space-separated values
