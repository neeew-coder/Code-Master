HTML Class Attribute

In this tutorial, you'll learn how to use the class attribute to style HTML elements, reuse styles across multiple tags, and interact with them using CSS and JavaScript.

🎨 Styling with a Class

Define a class in CSS using a period (.) followed by the class name:

<style>
  .city {
    background-color: tomato;
    color: white;
    border: 2px solid black;
    margin: 20px;
    padding: 20px;
  }
</style>>

🏙️ Applying the Class

Use the class attribute in HTML to apply the style:

<div class="city">
  <h2>London</h2>
  <p>London is the capital of England.</p>
</div>

📝 Styling Inline Elements

The class attribute works with inline elements too — like <span>: 

.note {
  font-size: 120%;
  color: red;
}
...
<h1>My <span class="note">Important</span> Heading</h1>

📚 Multiple Classes

You can assign more than one class to an element. Just separate them by spaces: 

<h2 class="city main">London</h2>

🔁 Shared Classes Across Elements

Different element types can share the same class:

<h2 class="city">Paris</h2>
<p class="city">Paris is the capital of France</p>

🧠 Using Class in JavaScript

You can target classes with JavaScript using getElementsByClassName(): 

function hideCities() {
  var x = document.getElementsByClassName("city");
  for (var i = 0; i < x.length; i++) {
    x[i].style.display = "none";
  }
}

📌 Syntax Reminder

  • Create a class in CSS using a period: .city { ... }
  • Add it to HTML using: class="city"
  • Class names are case sensitive
  • Multiple elements can share a class name
  • Multiple classes = space-separated values

🧠 Lesson Summary

  • class="button" assigns a reusable label to an element
  • class="btn primary" applies multiple styles
  • CSS targets classes using .classname selectors
  • Classes can be reused across multiple elements for consistent styling
