HTML Table Styling

CSS can be used to style HTML tables with colors, spacing, hover effects, and more — improving clarity and user engagement.

🎨 Zebra Striping (Horizontal)

Apply :nth-child(even) to <tr> to style every other row: 

tr:nth-child(even) {
  background-color: #D6EEEE;
}
1234
5678

📊 Zebra Striping (Vertical)

Apply :nth-child(even) to <td> and <th> to stripe columns: 

td:nth-child(even),
th:nth-child(even) {
  background-color: #D6EEEE;
}
1 2 3 4
5 6 7 8

🔀 Combined Striping

Use rgba() for transparent overlapping striping — vertically and horizontally: 

tr:nth-child(even) {
  background-color: rgba(150, 212, 212, 0.4);
}

th:nth-child(even),
td:nth-child(even) {
  background-color: rgba(150, 212, 212, 0.4);
}

📏 Horizontal Dividers

Add border-bottom to <tr> for line separation: 

tr {
  border-bottom: 1px solid #ddd;
}
First NameLast NameSavings
PeterGriffin$100
LoisGriffin$150

🖱️ Hoverable Rows

Use tr:hover to highlight rows on mouseover — ideal for interactive tables: 

tr:hover {
  background-color: #D6EEEE;
}
First NameLast NameSavings
PeterGriffin$100
LoisGriffin$150
JoeSwanson$300

📚 Summary

  • :nth-child(even) styles alternating rows or columns
  • rgba() allows transparent overlapping styles
  • border-bottom adds row dividers
  • tr:hover adds hover interactivity
