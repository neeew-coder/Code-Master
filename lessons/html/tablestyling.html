HTML Table Styling

CSS can be used to style HTML tables with colors, spacing, hover effects, and more — improving clarity and user engagement.

HTML Table Styling Tutorial demonstrates how to enhance table readability and interactivity using CSS — including zebra striping, hover effects, border styling, and spacing techniques.

📘 What You’ll Learn

  • How to apply zebra striping to alternate rows or columns using :nth-child()
  • How to use rgba() for layered transparency and subtle highlights
  • How to add horizontal dividers with border-bottom for visual separation
  • How to create interactive rows using tr:hover for hover feedback
  • How to combine multiple styling techniques for clean, engaging, and accessible tables

📊 Zebra Striping (Vertical)

Apply :nth-child(even) to <td> and <th> to stripe columns: 

td:nth-child(even),
th:nth-child(even) {
  background-color: #D6EEEE;
}
1 2 3 4
5 6 7 8

🔀 Combined Striping

Use rgba() for transparent overlapping striping — vertically and horizontally: 

tr:nth-child(even) {
  background-color: rgba(150, 212, 212, 0.4);
}

th:nth-child(even),
td:nth-child(even) {
  background-color: rgba(150, 212, 212, 0.4);
}

📏 Horizontal Dividers

Add border-bottom to <tr> for line separation: 

tr {
  border-bottom: 1px solid #ddd;
}

🖱️ Hoverable Rows

Use tr:hover to highlight rows on mouseover — ideal for interactive tables: 

tr:hover {
  background-color: #D6EEEE;
}

🧠 Lesson Summary

  • :nth-child(even) styles alternating rows or columns
  • rgba() allows transparent overlapping styles
  • border-bottom adds row dividers
  • tr:hover adds hover interactivity

🧪 Try It Yourself

Create a file named table-style.html. Add a table with three rows and four columns. Apply zebra striping to rows and columns, add hover effects, and use border-bottom for dividers. Bonus: try combining rgba() striping with hover transitions for a polished look.
