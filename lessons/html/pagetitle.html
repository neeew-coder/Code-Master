HTML Page Title

Every web page should have a descriptive title that summarizes the content and purpose of the page.

This tutorial explains how to use the <title> tag, where to place it inside the <head>, and how it affects browser tabs, bookmarks, and search engine visibility.

📘 What You’ll Learn

  • How to define a page title using <title> inside the <head> section
  • How the title appears in browser tabs and bookmarks
  • How search engines use the title for indexing and ranking
  • Best practices for writing clear, concise, and SEO-friendly titles

🏷️ The <title> Element

The <title> element is placed inside the <head> section. It is shown in the browser tab and used by search engines to index your page. 

<!DOCTYPE html>
<html>
<head>
  <title>HTML Tutorial</title>
</head>
<body>
  <p>The content of the document...</p>
</body>
</html>

(Browser tab title: HTML Tutorial)

✅ Why Page Titles Matter

  • Appears in browser tab and bookmarks
  • Helps search engines rank and display your page
  • Creates context when shared on social platforms

📈 SEO Tip

Choose a page title that is clear, concise, and relevant to your content. Avoid vague or generic phrases.

🧠 Lesson Summary

  • The <title> tag defines the page title shown in browser tabs
  • It must be placed inside the <head> section
  • Search engines use it to index and rank your page
  • It improves accessibility, bookmarking, and sharing
