HTML Links

HTML links (or hyperlinks) allow users to navigate between pages, documents, or resources. They’re essential for building connected web experiences.

🎥 Video Tutorial

Video by Alfa Technology – Watch on YouTube
This tutorial explains how to use <a> tags with href, target, title, mailto:, and image links — with clear examples and practical use cases.

📘 What You’ll Learn

  • How to create hyperlinks using <a href="">
  • How to open links in new tabs using target="_blank"
  • How to distinguish between absolute and relative URLs
  • How to make images and buttons clickable using anchor tags
  • How to create email links with mailto: and enhance accessibility using title

🔗 Basic Syntax

Use the <a> tag with the href attribute to define a hyperlink. 

<a href="https://codemaster.ph/">Visit CodeMaster!</a>
Visit CodeMaster!

🎯 The target Attribute

Use target="_blank" to open the link in a new tab or window. 

<a href="https://codemaster.ph/" target="_blank">Visit CodeMaster!</a>
Visit CodeMaster!

🌐 Absolute vs Relative URLs

Absolute URLs link to external sites. Relative URLs link within your own site. 

<h2>Absolute URLs</h2>
<p><a href="https://codemaster.ph/modules/html">HTML Module</a></p>
<p><a href="https://codemaster.ph/modules/css">CSS Module</a></p>

<h2>Relative URLs</h2>
<p><a href="html_intro.html">HTML Introduction</a></p>
<p><a href="/css/styles.html">CSS Styles</a></p>

🖼️ Image as a Link

Wrap an <img> tag inside an <a> tag to make the image clickable. 

<a href="course.html">
  <img src="icon.png" alt="Course link" style="width:42px;height:42px;">
</a>

📧 Email Link

Use mailto: in the href to open the user's email client. 

<a href="mailto:support@codemaster.ph">Send email</a>
Send email

🔘 Button as a Link

Use JavaScript to make a button behave like a link. 

<button onclick="document.location='course.html'">Go to Course</button>

🧠 Link Titles

Use the title attribute to show extra info on hover. 

<a href="https://codemaster.ph/html/" title="Explore HTML tutorials">
  Visit our HTML section
</a>
Visit our HTML section

🧠 Lesson Summary

  • Use <a> with href to create links
  • Use target="_blank" to open links in new tabs
  • Absolute URLs point to external sites; relative URLs point within your site
  • Wrap images or buttons in links to make them clickable
  • Use mailto: for email links
  • Use title to show extra info on hover
