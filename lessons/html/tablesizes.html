HTML Table Sizes

HTML tables can be sized individually at the table, row, or column level. To control sizing, use the style attribute with CSS properties like width and height.

📏 Table Width

Setting the table's width to 100% makes it span the full width of its parent element (usually the <body>). This helps the table adapt to different screen sizes. 

<table style="width:100%">
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
  </tr>
</table>
Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

📐 Column Width

Here, style="width:70%" is added to the first column header. That column now takes 70% of the entire table width. 

<table style="width:100%">
  <tr>
    <th style="width:70%">Firstname</th>
    <th>Lastname</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
  </tr>
</table>
Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

📏 Row Height

By adding style="height:200px" to the second row, it becomes visibly taller than others. 

<table style="width:100%">
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
    <th>Age</th>
  </tr>
  <tr style="height:200px">
    <td>Jill</td>
    <td>Smith</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
  </tr>
</table>
Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

🧪 Try It Yourself

Create a file named table-sizes.html. Add a table with three columns and three rows. Set the table width to 100%, make the first column 70% wide, and increase the height of the second row. Then test how the table behaves on different screen sizes. Bonus: try using CSS classes instead of inline styles. You can use our code playground to experiment with the examples above.
