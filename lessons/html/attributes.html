HTML Attributes

HTML attributes provide additional information about HTML elements. They’re always specified in the start tag and usually come in name/value pairs like name="value".

🔧 Attribute Basics

  • All HTML elements can have attributes
  • Attributes are written in the start tag
  • They usually follow the format: name="value"

🔗 The href Attribute

The <a> tag defines a hyperlink. The href attribute specifies the destination URL. 

<!DOCTYPE html>
<html>
  <body>
    <a href="new-page.html">New Page</a>
  </body>
</html>
  
<a href="new-page.html">New Page</a>

🖼️ The src Attribute

The <img> tag embeds an image. The src attribute specifies the image path. 

<!DOCTYPE html>
<html>
  <body>
    <img src="img_girl.jpg" alt="Girl">
  </body>
</html>
  
<img src="img_girl.jpg">

📏 Width and Height Attributes

Specify image dimensions in pixels: 

<!DOCTYPE html>
<html>
  <body>
    <img src="img_girl.jpg" width="500" height="600" alt="Girl">
  </body>
</html>
  
<img src="img_girl.jpg" width="500" height="600">

🧩 The alt Attribute

Provides alternate text if the image can’t be displayed: 

<!DOCTYPE html>
<html>
  <body>
    <img src="img_typo.jpg" alt="Girl with a jacket">
  </body>
</html>
  
<img src="img_typo.jpg" alt="Girl with a jacket">

🎨 The style Attribute

Used to apply inline CSS styles: 

<!DOCTYPE html>
<html>
  <body>
    <p style="color:red;">This is a red paragraph.</p>
  </body>
</html>
  
<p style="color:red;">This is a red paragraph.</p>

🌍 The lang Attribute

Declares the language of the page: 

<!DOCTYPE html>
<html lang="en-US">
  <body>
    <p>This page is declared in English (US).</p>
  </body>
</html>
  
<html lang="en-US">
  <body>
    <p>This page is declared in English (US).</p>
  </body>
</html>

💬 The title Attribute

Displays a tooltip when hovering over an element: 

<!DOCTYPE html>
<html>
  <body>
    <p title="I'm a tooltip">This is a paragraph.</p>
  </body>
</html>
  
<p title="I'm a tooltip">This is a paragraph.</p>

✅ Best Practices

  • Use lowercase attribute names
  • Always quote attribute values
  • Use double quotes unless single quotes are needed
