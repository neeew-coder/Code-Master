HTML Table Colspan & Rowspan

HTML tables support cells that span across multiple columns or rows. This allows for more flexible layouts and grouped content.

📐 Colspan (Across Columns)

To make a cell stretch across multiple columns, use the colspan attribute inside a <th> or <td>. The value represents how many columns to span. 

<table>
  <tr>
    <th colspan="2">Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>43</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>57</td>
  </tr>
</table>
Name Age
Jill Smith 43
Eve Jackson 57

📏 Rowspan (Down Rows)

To make a cell stretch over multiple rows, use the rowspan attribute. This is helpful when labeling vertically stacked data. 

<table>
  <tr>
    <th>Name</th>
    <td>Jill</td>
  </tr>
  <tr>
    <th rowspan="2">Phone</th>
    <td>555-1234</td>
  </tr>
  <tr>
    <td>555-8745</td>
  </tr>
</table>
Name Jill
Phone 555-1234
555-8745

📚 Summary

  • colspan allows a cell to stretch across multiple columns
  • rowspan allows a cell to stretch across multiple rows
  • Use with <td> or <th> elements
