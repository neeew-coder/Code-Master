HTML Forms

HTML forms are used to collect user input. This input is typically sent to a server for processing. Forms contain fields like text boxes, radio buttons, checkboxes, and submit buttons.

üìÑ The <form> Element

The <form> element wraps all input fields. It defines how data is grouped and where it will be sent. The action attribute specifies the server endpoint, and method defines how data is sent (e.g. GET or POST). 

<form action="/submit.php" method="post">
  ...form elements...
</form>

üì§ Submit Button

Use <input type="submit"> to send data to a server. The action attribute defines the endpoint. 






<form action="/action_page.php">
  <label for="fname">First name:</label><br>
  <input type="text" id="fname" name="fname" value="John"><br>
  
  <label for="lname">Last name:</label><br>
  <input type="text" id="lname" name="lname" value="Doe"><br><br>
  
  <input type="submit" value="Submit">
</form>

Note: Inputs must have a name attribute to be submitted.

üß† Lesson Summary

  • <form> wraps all input fields and defines submission behavior
  • <input type="text"> creates single-line text fields
  • <input type="radio"> allows one selection from a group
  • <input type="checkbox"> allows multiple selections
  • <input type="submit"> sends data to the server
