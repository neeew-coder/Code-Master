HTML Lists

HTML supports three types of lists: unordered, ordered, and description. Each serves a different purpose and helps structure content clearly.

📋 Unordered Lists

Use <ul> for unordered lists and <li> for each item. The default marker is a bullet. 

<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>
  • Coffee
  • Tea
  • Milk

🪆 Nested Lists

<ul>
  <li>Coffee</li>
  <li>Tea
    <ul>
      <li>Black tea</li>
      <li>Green tea</li>
    </ul>
  </li>
  <li>Milk</li>
</ul>

  • Tea
    • Black tea
    • Green tea
  • Milk

📐 Horizontal List (Navigation)

ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
  overflow: hidden;
  background-color: #333;
}
li { float: left; }
li a {
  display: block;
  color: white;
  padding: 16px;
}
li a:hover {
  background-color: #111;
}

<ul>
  <li>Home</li>
  <li>News</li>
  <li>Contact</li>
  <li>About</li>
</ul>
  • Home
  • News
  • Contact
  • About

🔢 Ordered Lists

Use <ol> for ordered lists. type controls numbering style: 

<ol type="A">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>
  1. Coffee
  2. Tea
  3. Milk
<ol start="50">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>
  1. Coffee
  2. Tea
  3. Milk

📘 HTML List Tags Reference

TagPurpose
<ul>Unordered list
<ol>Ordered list
<li>List item
<dl>Description list
<dt>Term in description list
<dd>Term description

📚 Chapter Summary

  • <ul> creates an unordered bullet list
  • <ol> creates a numbered list
  • <li> defines a list item
  • <dl> creates a description list
  • <dt> adds a term, and <dd> defines it
  • Use list-style-type to change unordered markers
  • Use type and start for ordered list control
  • Lists can be nested, styled, or even used for horizontal navigation
