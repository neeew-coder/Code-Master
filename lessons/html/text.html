HTML Text Formatting

In this tutorial, you'll learn how to format text in HTML using tags like <b>, <i>, <mark>, <del>, and more. These tags help convey meaning and improve readability.

📘 What You’ll Learn

  • How to emphasize text with <strong> and <em>
  • How to highlight with <mark> and shrink with <small>
  • How to format inline code with <code>
  • How to use <sup> and <sub> for scientific notation

🔤 Formatting Elements

  • <b> — Bold text
  • <strong> — Important text
  • <i> — Italic text
  • <em> — Emphasized text
  • <mark> — Highlighted text
  • <small> — Smaller text
  • <del> — Deleted text
  • <ins> — Inserted text
  • <sub> — Subscript text
  • <sup> — Superscript text

🟡 Bold and Strong

<b>This text is bold</b>
<strong>This text is important!</strong>
This text is bold
This text is important!

🟣 Italic and Emphasized

<i>This text is italic</i>
<em>This text is emphasized</em>
This text is italic
This text is emphasized

🔍 Small Text

<small>This is some smaller text.</small>
This is some smaller text.

🟨 Marked Text

<p>Do not forget to buy <mark>milk</mark> today.</p>

Do not forget to buy milk today.

❌ Deleted Text

<p>My favorite color is <del>blue</del> red.</p>

My favorite color is blue red.

✅ Inserted Text

<p>My favorite color is <del>blue</del> <ins>red</ins>.</p>

My favorite color is blue red.

🧪 Subscript and Superscript

<p>This is <sub>subscripted</sub> text.</p>
<p>This is <sup>superscripted</sup> text.</p>

This is subscripted text.

This is superscripted text.

🧠 Lesson Summary

  • <strong> indicates importance; <em> adds emphasis
  • <mark> highlights text visually
  • <code> formats technical snippets
  • <sup> and <sub> position text above or below baseline
  • <small> reduces font size for disclaimers or notes
