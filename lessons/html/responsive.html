Responsive Web Design

Responsive design ensures websites adapt smoothly across devices. It uses flexible layouts, scalable units, and media queries to create mobile-friendly, accessible experiences.

üì± Key Concepts

  • <meta name="viewport"> tells the browser how to scale the page
  • Use percentages, vw, rem, and em for scalable sizing
  • Use media queries to change styles based on screen size
  • Flexbox and Grid help create adaptable layouts

üîç Viewport Meta Tag

<meta name="viewport" content="width=device-width, initial-scale=1.0">

This helps the browser render your layout properly on mobile screens.

üìÑ responsive.html Source Code

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Responsive Layout</title>
  <style>
    body {
      font-family: Arial, sans-serif;
      padding: 20px;
    }
    .container {
      display: flex;
      gap: 20px;
    }
    .item {
      flex: 1;
      padding: 20px;
      background-color: #eee;
      border-radius: 8px;
    }
    @media (max-width: 600px) {
      .container {
        flex-direction: column;
      }
    }
  </style>
</head>
<body>

  <h1>Responsive Cities</h1>
  <div class="container">
    <div class="item">
      <h2>London</h2>
      <p>Capital of England</p>
    </div>
    <div class="item">
      <h2>Paris</h2>
      <p>Capital of France</p>
    </div>
    <div class="item">
      <h2>Tokyo</h2>
      <p>Capital of Japan</p>
    </div>
  </div>

</body>
</html>
