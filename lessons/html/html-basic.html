HTML Basic Examples

In this chapter we’ll explore some basic HTML examples. Don’t worry if we use tags you haven’t learned yet — you’ll get there!

📄 HTML Documents

All HTML documents must start with a document type declaration: <!DOCTYPE html>.

The document begins with <html> and ends with </html>. The visible part is between <body> and </body>. 

<html>
  <body>

    <h1>First Heading</h1>
    <p>first paragraph.</p>

  </body>
</html>

📌 The <!DOCTYPE> Declaration

  • Represents the document type
  • Helps browsers display pages correctly
  • Must appear once, at the top of the page
  • Not case sensitive
  • HTML5 uses: <!DOCTYPE html>

🔠 HTML Headings

Headings are defined with <h1> to <h6>. <h1> is the most important, <h6> the least. 

This is heading 1


This is heading 2


This is heading 3

📄 HTML Paragraphs

Paragraphs are defined with the <p> tag. 

    <p>this is a paragraph</p>
    <p>this is another paragraph.</p>

🔗 HTML Links

Links are defined with the <a> tag. The destination is set using the href attribute. 

<a href="https://www.example.com">This is a link</a>

🖼️ HTML Images

Images are defined with the <img> tag. Attributes like src, alt, width, and height provide details. 

<img src="image.jpg" alt="Description" width="300" height="200">

🔍 How to View HTML Source

  • Press CTRL + U or right-click and choose “View Page Source”
  • Use “Inspect” to view and edit HTML/CSS live in the browser
