ğŸ“„ HTML Documents

All HTML documents must start with a document type declaration: <!DOCTYPE html>.

The document begins with <html> and ends with </html>. The visible part is between <body> and </body>. 

<html>
  <body>

    <h1>First Heading</h1>
    <p>First paragraph.</p>

  </body>
</html>

ğŸ“Œ The <!DOCTYPE> Declaration

  • Represents the document type
  • Helps browsers display pages correctly
  • Must appear once, at the top of the page
  • Not case sensitive
  • HTML5 uses: <!DOCTYPE html>
