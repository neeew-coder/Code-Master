HTML Table Borders

HTML tables can be styled with borders to better organize and emphasize data. With CSS, you control how those borders appear and behave.

HTML Table Borders Tutorial explains how to style table borders using both HTML and CSS — including border-collapse, border-style, border-radius, and border-color.

📘 What You’ll Learn

  • How to add borders to tables using CSS properties
  • How to collapse borders between adjacent cells with border-collapse
  • How to style borders using border-style (solid, dashed, dotted)
  • How to apply rounded corners with border-radius and hide borders selectively
  • How to customize border color and apply styles to specific cells or rows

🧱 Adding a Basic Border

Apply the border property to table elements:

table, th, td {
  border: 1px solid black;
}

🧊 Collapsing Borders

Use border-collapse: collapse to remove double borders between cells: 

table, th, td {
  border: 1px solid black;
  border-collapse: collapse;
}

👻 Invisible Borders

White borders on a colored background give a seamless look: 

table, th, td {
  border: 1px solid white;
  border-collapse: collapse;
}

th, td {
  background-color: #96D4D4;
}

🏵️ Rounded Corners

Use border-radius to round table borders:

table, th, td {
  border: 1px solid black;
  border-radius: 10px;
}

🌿 Selective Borders

Skip the border around the full table:

th, td {
  border: 1px solid black;
  border-radius: 10px;
}

🎯 Border Styles

You can change how borders appear with border-style: 

th, td {
  border-style: dotted;
}

Allowed styles include: dotted, dashed, solid, double, groove, ridge, inset, outset, none, and hidden.

🌈 Border Color

Set border colors using border-color:

th, td {
  border-color: orange;
}

📚 Summary

  • border adds a table border
  • border-collapse merges inner lines
  • border-style controls the design (dotted, dashed, solid...)
  • border-color defines border color
  • border-radius adds rounded corners
