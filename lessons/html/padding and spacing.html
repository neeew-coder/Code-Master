HTML Table Padding & Spacing

HTML tables can control how data looks inside and between cells. Use padding to adjust space inside cells, and spacing to create gaps between cells.

🔳 Cell Padding

Padding is the space between the cell border and the cell content. By default, cell padding is 0. To add padding, use the padding property on th and td: 

th, td {
  padding: 15px;
}

📐 Padding Per Side

To control padding on specific sides of a cell, use directional properties: 

th, td {
  padding-top: 10px;
  padding-bottom: 20px;
  padding-left: 30px;
  padding-right: 40px;
}

🧱 Cell Spacing

Spacing is the gap between cells. To adjust it, apply the border-spacing property to the table element. 

table {
  border-spacing: 30px;
}
