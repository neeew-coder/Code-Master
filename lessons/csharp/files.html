📁 C# File Handling

File handling in C# allows you to read from, write to, and manage files using the System.IO namespace. It’s essential for data persistence, logging, and configuration.

📘 What You'll Learn
  • Reading and writing text files
  • Appending content
  • Checking file existence
  • Using File, StreamReader, and StreamWriter

🔍 Explanation
  • File.WriteAllText: Overwrites file with new content
  • File.AppendAllText: Adds content to the end
  • File.ReadAllText: Reads entire file as string
  • File.Exists: Checks if file is present

🧠 Analogy: File handling is like writing in a notebook — you can start fresh, add notes, or read what’s already written.

📚 More Examples

Using StreamWriter:
using (StreamWriter sw = new StreamWriter("log.txt")) {
  sw.WriteLine("Log entry");
}

Using StreamReader:
using (StreamReader sr = new StreamReader("log.txt")) {
  string line = sr.ReadToEnd();
}

💡 Tips: Always check file existence before reading. Use using blocks for stream safety. Avoid hardcoded paths in production.
