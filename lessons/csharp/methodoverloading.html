🔁 C# Method Overloading

Method overloading allows multiple methods with the same name but different parameter lists. It improves flexibility and readability by tailoring behavior to different inputs.

🔍 Explanation
  • All methods are named Greet
  • Each has a different parameter signature
  • The compiler selects the correct method based on arguments
  • Return type alone is not enough to overload

🧠 Analogy: Overloading is like having one contact name in your phone but choosing how to message them — text, call, or email — based on context.

📚 More Examples

Overload by type:
static int Multiply(int a, int b) => a * b;
static double Multiply(double a, double b) => a * b;

Overload by parameter count:
static void Log(string message);
static void Log(string message, int level);

💡 Tips: Keep overloads intuitive. Avoid excessive overloads that confuse intent. Use optional parameters when appropriate.
