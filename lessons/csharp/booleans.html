📄 Boolean Syntax

🔍 Explanation
  • ==: Equal to
  • !=: Not equal to
  • &&: Logical AND
  • ||: Logical OR
  • !: Logical NOT

🧠 Analogy: Booleans are like light switches — true means ON, false means OFF. You combine switches to control what happens next.

📚 More Examples

Comparison:
bool isEqual = (5 == 5); // true
bool isNotEqual = (5 != 3); // true

Logical combination:
bool result = (true || false) && !false; // true

💡 Tips: Use parentheses to group complex boolean expressions for clarity and correct evaluation order.

🧠 Lesson Summary

The bool type in C# stores either true or false. It’s essential for controlling program logic, evaluating conditions, and building interactive flows. This tutorial shows how booleans work with comparisons, logical operators, and control structures — forming the backbone of decision-making in C#.

🧪 Try It Yourself

// BooleanDemo.cs
using System;

class BooleanDemo {
  static void Main() {
    bool isLoggedIn = true;
    bool hasAccess = false;

    if (isLoggedIn && hasAccess) {
      Console.WriteLine("Welcome to the system.");
    } else {
      Console.WriteLine("Access denied.");
    }

    Console.WriteLine("Negation: " + !isLoggedIn);
  }
}
