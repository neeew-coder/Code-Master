🔘 C# Booleans

Booleans represent truth values: true or false. They're essential for decision-making in code, especially in conditions and loops. This module covers boolean variables, comparisons, and logical operators.

  • Using comparison operators

📄 Boolean Syntax

🔍 Explanation
  • ==: Equal to
  • !=: Not equal to
  • &&: Logical AND
  • ||: Logical OR
  • !: Logical NOT

🧠 Analogy: Booleans are like light switches — true means ON, false means OFF. You combine switches to control what happens next.

📚 More Examples

Comparison:
bool isEqual = (5 == 5); // true
bool isNotEqual = (5 != 3); // true

Logical combination:
bool result = (true || false) && !false; // true

💡 Tips: Use parentheses to group complex boolean expressions for clarity and correct evaluation order.
