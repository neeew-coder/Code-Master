🧩 C# Methods

Methods are reusable blocks of code that perform specific tasks. They help organize logic, reduce repetition, and improve readability.

📘 What You'll Learn
  • Declaring and calling methods
  • Using parameters and return values
  • Understanding method scope
  • Best practices for modular design

🔍 Explanation
  • static void GreetUser(string name): Declares a method with a parameter
  • static int Add(int a, int b): Returns an integer result
  • Methods can be reused and composed for modular logic

🧠 Analogy: A method is like a recipe — you give it ingredients (parameters), follow steps (logic), and get a result (return value).

📚 More Examples

Method with multiple parameters:
static double Multiply(double x, double y) {
  return x * y;
}

Void method:
static void ShowMessage() {
  Console.WriteLine("Welcome!");
}

💡 Tips: Use descriptive names. Keep methods focused on one task. Avoid long methods — break them into smaller ones.
