Methods

Methods are reusable blocks of code that perform specific tasks. They help organize logic and reduce repetition in C# programs.

🎯 Learning Objectives

  • Define and call methods
  • Use parameters and return values

🧩 Key Concepts

  • Method signatures and scope
  • Return types and void methods

🛠️ Real-world Application

Methods are used for calculations, event handling, and modular programming.

✅ Expected Outcome

You’ll be able to write and use methods to structure your C# programs. 

static string Greet(string name) {
  return $"Hello, {name}!";
}

Console.WriteLine(Greet("Mark"));
