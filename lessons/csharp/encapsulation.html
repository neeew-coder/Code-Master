🛡️ C# Encapsulation

Encapsulation is the practice of hiding internal data and exposing only what’s necessary. It helps maintain control, prevent misuse, and enforce consistency in object behavior.

🧠 Lesson Summary

Encapsulation in C# is the practice of restricting direct access to class members and exposing controlled interfaces. This tutorial shows how to use private fields, public properties, and validation logic to protect data integrity and simplify class usage. It’s a core principle of object-oriented programming.

🧪 Try It Yourself

// EncapsulationDemo.cs
using System;

class BankAccount {
  private decimal balance;

  public decimal Balance {
    get { return balance; }
    set {
      if (value >= 0) balance = value;
    }
  }

  public void Deposit(decimal amount) {
    if (amount > 0) balance += amount;
  }
}

class Program {
  static void Main() {
    BankAccount account = new BankAccount();
    account.Deposit(1000);
    Console.WriteLine("Balance: " + account.Balance);
  }
}
