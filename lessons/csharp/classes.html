Classes & Objects

C# is an object-oriented language. Classes define blueprints for objects, which encapsulate data and behavior.

🎯 Learning Objectives

  • Define classes and create objects
  • Use constructors and methods

🧩 Key Concepts

  • Encapsulation and access modifiers
  • Object instantiation

🛠️ Real-world Application

Classes and objects are used in everything from game characters to data models in enterprise apps.

✅ Expected Outcome

You’ll be able to define classes and create objects to organize your code. 

class Person {
  public string Name;

  public Person(string name) {
    Name = name;
  }

  public void Greet() {
    Console.WriteLine($"Hello, {Name}!");
  }
}

Person p = new Person("Mark");
p.Greet();
