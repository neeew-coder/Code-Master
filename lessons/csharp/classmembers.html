🧩 C# Class Members

Class members are the building blocks of a class. They include fields for storing data, properties for controlled access, methods for behavior, and constructors for initialization.

  • Fields vs properties
  • Constructors and method members
  • Access modifiers: public, private, protected

🔍 Explanation
  • Field: brand stores internal data

🧠 Analogy: Think of a class as a machine. Fields are internal parts, properties are knobs you can turn, methods are actions it performs, and constructors are how you assemble it.

Private field with public property:

💡 Tips: Use properties instead of public fields. Group members logically: fields first, then constructors, then methods. Use access modifiers to control visibility.
