üîç Explanation
  • Math.Pow(x, y): Raises x to the power of y
  • Math.Sqrt(x): Returns the square root of x
  • Math.Abs(x): Returns the absolute value of x
  • Math.Round(x): Rounds x to the nearest whole number
  • Math.Ceiling(x): Rounds x up
  • Math.Floor(x): Rounds x down

üß™ Analogy: The Math class is like a scientific calculator ‚Äî it gives you tools to go beyond basic arithmetic and solve more complex problems with precision.

üß™ Try It Yourself

// MathDemo.cs
using System;

class MathDemo {
  static void Main() {
    double x = 5.7;
    Console.WriteLine("Round: " + Math.Round(x));
    Console.WriteLine("Ceiling: " + Math.Ceiling(x));
    Console.WriteLine("Floor: " + Math.Floor(x));
    Console.WriteLine("Power: " + Math.Pow(2, 3));
    Console.WriteLine("Square Root: " + Math.Sqrt(49));
    Console.WriteLine("Absolute: " + Math.Abs(-10));
    Console.WriteLine("Max: " + Math.Max(10, 20));
    Console.WriteLine("Min: " + Math.Min(10, 20));
  }
}
