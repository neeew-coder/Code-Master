📐 C# Math Operations

C# provides built-in mathematical operators and functions through the Math class. You can perform basic arithmetic, rounding, powers, roots, and trigonometric calculations. This module introduces the most common math operations used in programming.

📘 What You'll Learn
  • Basic arithmetic operations
  • Using the Math class for advanced calculations
  • Rounding and absolute value functions
  • Working with powers and square roots

🔍 Explanation
  • Math.Pow(x, y): Raises x to the power of y
  • Math.Sqrt(x): Returns the square root of x
  • Math.Abs(x): Returns the absolute value of x
  • Math.Round(x): Rounds x to the nearest whole number
  • Math.Ceiling(x): Rounds x up
  • Math.Floor(x): Rounds x down

🧪 Analogy: The Math class is like a scientific calculator — it gives you tools to go beyond basic arithmetic and solve more complex problems with precision.

📚 More Examples

Trigonometry:
double angle = Math.PI / 2;
double sine = Math.Sin(angle); // Result: 1

Min and Max:
int min = Math.Min(5, 10); // Result: 5
int max = Math.Max(5, 10); // Result: 10

💡 Tips: Use Math.Round() with a second argument to control decimal precision. For example: Math.Round(3.14159, 2) returns 3.14.
