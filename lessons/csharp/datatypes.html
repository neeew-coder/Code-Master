🔢 C# Data Types

Data types define the kind of data a variable can store. C# is a statically typed language, so you must specify the type when declaring a variable. Choosing the right type helps your program run efficiently and correctly.

🔍 Explanation
  • int: Whole numbers (e.g., 1, 42)
  • float: Decimal numbers with less precision
  • double: Decimal numbers with higher precision
  • char: A single character (e.g., 'A')
  • string: A sequence of characters (e.g., "Hello")
  • bool: Boolean values (true or false)

🧪 Analogy: Think of data types like different kinds of containers: a cup for water (float), a jar for cookies (int), a box for letters (string). You choose the container based on what you're storing.

💡 Tips: Use float for small decimals, double for precise calculations, and string for text. Always match the type to the data you're working with.
