Introduction to C#

C# (C-Sharp) is a modern, object-oriented programming language developed by Microsoft. It’s widely used for building desktop apps, games, and web services with .NET.

🎯 Learning Objectives

  • Understand what C# is and where it’s used
  • Write and run a basic C# program

🧩 Key Concepts

  • C# syntax and structure
  • Compilation and execution in .NET

🛠️ Real-world Application

C# powers Windows apps, Unity games, and enterprise systems built on .NET.

✅ Expected Outcome

You’ll be able to write a simple C# program and understand its structure. 

using System;

class Program {
  static void Main() {
    Console.WriteLine("Hello, Mark!");
  }
}
