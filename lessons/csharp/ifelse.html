🔀 C# If / Else

Conditional statements allow your program to make decisions based on logic. In C#, you use if, else if, and else to control which blocks of code run depending on conditions.

🔍 Explanation
  • if: Executes code if the condition is true
  • else if: Adds additional conditions
  • else: Executes if no previous conditions matched
  • Conditions must evaluate to bool

🧠 Analogy: Think of if / else like a fork in the road — your program chooses a path based on the condition. If one path doesn’t match, it tries the next until it finds a match or defaults.

📚 More Examples

Simple check:
int age = 18;
if (age >= 18) { Console.WriteLine("Adult"); }

Multiple branches:
string role = "admin";
if (role == "user") {...}
else if (role == "admin") {...}
else {...}

💡 Tips: Keep conditions simple and readable. Use indentation and braces even for single-line blocks to avoid logic errors.
