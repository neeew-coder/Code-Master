🧾 C# Input Validation

Validating user input ensures your program behaves correctly and avoids crashes. This module demonstrates how to safely convert user input using int.TryParse() and handle invalid cases gracefully.

📘 What You'll Learn
  • How to retrieve user input via placeholders
  • Safely convert strings to integers
  • Display formatted output
  • Handle invalid input using conditional logic

📄 Input Validation Syntax

Invalid age entered.
🔍 Explanation
  • TryParse(): Safely converts a string to an integer
  • Conditional logic: Checks if conversion was successful
  • Output formatting: Uses string interpolation with $""

🧪 Analogy: Validating input is like checking ingredients before cooking. If something’s missing or wrong, you don’t proceed — you fix it first.

📚 More Examples

Invalid input:
string ageInput = "twenty";
bool success = int.TryParse(ageInput, out int age); // Result: false

Valid input:
string ageInput = "22";
bool success = int.TryParse(ageInput, out int age); // Result: true

💡 Tips: Always use TryParse() when working with user input. It prevents crashes and lets you handle errors gracefully.
