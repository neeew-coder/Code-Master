📚 More Examples

Invalid input:
string ageInput = "twenty";
bool success = int.TryParse(ageInput, out int age); // Result: false

Valid input:
string ageInput = "22";
bool success = int.TryParse(ageInput, out int age); // Result: true

💡 Tips: Always use TryParse() when working with user input. It prevents crashes and lets you handle errors gracefully.

🧠 Lesson Summary

C# uses Console.ReadLine() to capture user input from the terminal. Since input is read as a string, it often needs to be converted to other types using int.Parse(), Convert.ToDouble(), or similar methods. This tutorial shows how to build interactive programs that respond to user data.

🧪 Try It Yourself

// UserInputDemo.cs
using System;

class UserInputDemo {
  static void Main() {
    Console.Write("Enter your name: ");
    string name = Console.ReadLine();

    Console.Write("Enter your age: ");
    int age = int.Parse(Console.ReadLine());

    Console.WriteLine("Hello, " + name + "! You are " + age + " years old.");
  }
}
