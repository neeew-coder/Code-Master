🧵 C# Strings

Strings in C# are sequences of characters used to represent text. The string type comes with powerful methods for manipulation, comparison, and formatting. This module introduces string basics and common operations.

📘 What You'll Learn
  • Declaring and initializing strings
  • Concatenation and interpolation
  • Common string methods like Length, ToUpper(), Contains()
  • Trimming, replacing, and splitting strings

🔍 Explanation
  • name.Length: Returns the number of characters
  • ToUpper(): Converts to uppercase
  • Contains(): Checks if a substring exists
  • Trim(): Removes leading/trailing whitespace
  • Replace(): Substitutes part of the string
  • Split(): Breaks string into array using delimiter

🧶 Analogy: Think of strings like beads on a necklace — each character is a bead, and string methods let you rearrange, count, or swap them to create new patterns.

📚 More Examples

Escape characters:
string path = "C:\\Users\\Mark\\Documents";
string quote = "He said, \"Hello!\"";

Substring:
string word = "Programming";
string part = word.Substring(0, 4); // "Prog"

💡 Tips: Use $"..." for cleaner string interpolation. Avoid using + for multiple concatenations — it’s less readable and less efficient.
