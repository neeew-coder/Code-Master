🧮 C# Arrays

Arrays are fixed-size collections of elements of the same type. They're indexed starting at 0 and allow fast access and iteration.

📄 Array Syntax

🔍 Explanation
  • int[] numbers = { 10, 20, 30 }: Declares and initializes
  • numbers[2] = 35: Modifies the third element
  • numbers.Length: Gets array size
  • Use for or foreach to iterate

🧠 Analogy: Think of an array like a row of mailboxes — each with a number and a fixed slot for one item.

📚 More Examples

Empty array:
string[] names = new string[3];
names[0] = "Alice";

Foreach loop:
foreach (int num in numbers) {
  Console.WriteLine(num);
}

💡 Tips: Arrays are fixed in size. For dynamic collections, use List<T>. Always check bounds to avoid IndexOutOfRangeException.

🧠 Lesson Summary

Arrays in C# are fixed-size collections of elements of the same type, accessed by index. This tutorial shows how to declare arrays, populate them, and iterate through them using loops. Arrays are ideal for storing ordered data like scores, names, or settings — and serve as a gateway to more advanced collections.

🧪 Try It Yourself

// ArrayDemo.cs
using System;

class ArrayDemo {
  static void Main() {
    string[] fruits = { "Apple", "Banana", "Cherry" };

    for (int i = 0; i < fruits.Length; i++) {
      Console.WriteLine("Fruit #" + (i + 1) + ": " + fruits[i]);
    }

    Console.WriteLine("Using foreach:");
    foreach (string fruit in fruits) {
      Console.WriteLine(fruit);
    }
  }
}
