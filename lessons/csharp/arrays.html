🧮 C# Arrays

Arrays are fixed-size collections of elements of the same type. They're indexed starting at 0 and allow fast access and iteration.

📘 What You'll Learn
  • How to declare and initialize arrays
  • Accessing and modifying array elements
  • Iterating through arrays
  • Common array operations

📄 Array Syntax

🔍 Explanation
  • int[] numbers = { 10, 20, 30 }: Declares and initializes
  • numbers[2] = 35: Modifies the third element
  • numbers.Length: Gets array size
  • Use for or foreach to iterate

🧠 Analogy: Think of an array like a row of mailboxes — each with a number and a fixed slot for one item.

📚 More Examples

Empty array:
string[] names = new string[3];
names[0] = "Alice";

Foreach loop:
foreach (int num in numbers) {
  Console.WriteLine(num);
}

💡 Tips: Arrays are fixed in size. For dynamic collections, use List<T>. Always check bounds to avoid IndexOutOfRangeException.
