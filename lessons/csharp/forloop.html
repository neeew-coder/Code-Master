🔁 C# For Loop

The for loop is ideal for repeating actions a specific number of times. It uses a counter variable, a condition, and an increment step — all in one compact line.

  • How for loops work
  • How to control iteration with counters
  • How to loop through ranges and arrays

📄 For Loop Syntax

Output will appear here.
🔍 Explanation
  • int i = 0: Initializes the counter
  • i < 5: Condition checked before each loop
  • i++: Increments counter after each iteration
  • Use i-- for reverse loops

🧠 Analogy: A for loop is like setting a timer — you know how many ticks you want, and it counts up or down until done.

📚 More Examples

Even numbers:
for (int i = 0; i <= 10; i += 2) { Console.WriteLine(i); }

Nested loop:
for (int row = 1; row <= 3; row++) {
  for (int col = 1; col <= 3; col++) {
    Console.WriteLine($"Cell {row},{col}");
  }
}

💡 Tips: Use for when you know the exact number of iterations. For unknown counts, prefer while.
