🔄 C# While & Do-While Loops

Loops allow your program to repeat actions while a condition remains true. The while loop checks the condition before each iteration, while do-while guarantees at least one execution.

🔍 Explanation
  • while: Checks condition before each loop
  • do-while: Executes once before checking condition
  • count++: Increments to avoid infinite loop
  • Use break or return to exit early if needed

🧠 Analogy: A while loop is like checking if the door is open before entering. A do-while loop is like walking in first, then checking if you should stay.

📚 More Examples

Input validation:
while (!isValid) { PromptUser(); }

Guaranteed execution:
do { ShowMenu(); } while (!exit);

💡 Tips: Always update your loop variable. Infinite loops can crash your program or consume resources.
