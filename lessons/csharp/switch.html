🔁 C# Switch Statement

The switch statement provides a streamlined way to handle multiple conditions based on a single variable. It’s ideal when comparing discrete values like enums, integers, or strings.

🔍 Explanation
  • switch: Evaluates a variable once
  • case: Matches specific values
  • break: Prevents fall-through to next case
  • default: Handles unmatched values

🧠 Analogy: Think of switch like a menu — you choose one item, and the program serves the matching response. If your choice isn’t listed, you get the default dish.

📚 More Examples

Enum switch:
enum Status { Ready, Waiting, Error }
Status current = Status.Waiting;
switch (current) { ... }

Fall-through prevention:
Always use break unless intentional fall-through is needed.

💡 Tips: Use switch when comparing a single variable to multiple discrete values. Avoid complex conditions — use if for those.
