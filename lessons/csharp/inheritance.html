🧬 C# Inheritance

Inheritance allows one class to acquire the properties and behaviors of another. It promotes code reuse, logical hierarchy, and polymorphic behavior.

📘 What You'll Learn
  • Base and derived class relationships
  • Using : base syntax
  • Method overriding with virtual and override
  • Accessing base class members

🔍 Explanation
  • virtual: Marks a method as overridable
  • override: Replaces base method behavior
  • base: Refers to the parent class
  • Inheritance uses : to link classes

🧠 Analogy: Inheritance is like a family tree — children inherit traits from parents but can also develop their own.

📚 More Examples

Calling base method:
public override void Speak() { base.Speak(); Console.WriteLine("Dog barks"); }

Multi-level inheritance:
class Mammal : Animal {}
class Cat : Mammal {}

💡 Tips: Use inheritance for shared behavior. Prefer composition when relationships are not hierarchical. Avoid deep inheritance chains.
