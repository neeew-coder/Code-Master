📤 C# Output Basics

This module covers how to display information in C# using the Console.WriteLine() method. Output is essential for debugging, user interaction, and showing results.

📘 What You'll Learn
  • Using Console.WriteLine() for output
  • Printing variables and expressions
  • Formatting output with strings
  • Difference between WriteLine() and Write()

📄 Output Example

Output will appear here.
🔍 Explanation
  • Console.WriteLine() prints a line and moves to the next.
  • Console.Write() prints without a newline.
  • Use + to concatenate strings and variables.
  • Output helps visualize program behavior.

🧠 Analogy: Think of output like a speaker in a room. It broadcasts messages to the audience (users) so they can see what's happening in the program.

📚 More Examples

String Formatting:
Console.WriteLine("Name: {0}, Age: {1}", "Mark", 30);

Newline and Tabs:
Console.WriteLine("Line1\nLine2\tTabbed");

💡 Tips: Use \n for new lines and \t for tabs. Prefer WriteLine() for readable output unless you're building dynamic inline text.
