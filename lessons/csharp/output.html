🧠 Analogy: Think of output like a speaker in a room. It broadcasts messages to the audience (users) so they can see what's happening in the program.

📚 More Examples

String Formatting:
Console.WriteLine("Name: {0}, Age: {1}", "Mark", 30);

Newline and Tabs:
Console.WriteLine("Line1\nLine2\tTabbed");

💡 Tips: Use \n for new lines and \t for tabs. Prefer WriteLine() for readable output unless you're building dynamic inline text.

🧠 Lesson Summary

C# uses Console.WriteLine() and Console.Write() to display output in the terminal. WriteLine() appends a newline after printing, while Write() keeps the cursor on the same line. These methods are essential for debugging, prompting user input, and displaying results in CLI applications.
