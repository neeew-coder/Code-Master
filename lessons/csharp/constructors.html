🏗️ C# Constructors

Constructors are special methods used to initialize objects when they’re created. They can accept parameters, set default values, and even call other constructors using chaining.

📘 What You'll Learn
  • Default and parameterized constructors
  • Constructor overloading
  • Using this for chaining
  • Best practices for initialization

🔍 Explanation
  • Default constructor: No parameters, sets defaults
  • Parameterized constructor: Accepts values
  • Chained constructor: Uses this(...) to reuse logic
  • Constructors run automatically when new is used

🧠 Analogy: A constructor is like a setup wizard — it configures your object before you start using it.

📚 More Examples

Overloaded constructors:
public User(string name);
public User(string name, int age);

Constructor chaining:
public User(string name) : this(name, 0) {}

💡 Tips: Use constructor chaining to avoid duplication. Always ensure required fields are initialized. Avoid heavy logic inside constructors.
