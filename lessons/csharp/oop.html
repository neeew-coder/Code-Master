🧱 C# Object-Oriented Programming (OOP)

OOP is a programming paradigm based on the concept of "objects" — reusable units that encapsulate data and behavior. C# supports OOP through classes, interfaces, and inheritance.

📘 What You'll Learn
  • Creating and using classes and objects
  • Encapsulation with fields and properties
  • Inheritance and base classes
  • Polymorphism and method overriding
  • Abstraction via interfaces and abstract classes
