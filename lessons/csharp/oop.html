🧱 C# Object-Oriented Programming (OOP)

OOP is a programming paradigm based on the concept of "objects" — reusable units that encapsulate data and behavior. C# supports OOP through classes, interfaces, and inheritance.

🧠 Lesson Summary

Object-oriented programming (OOP) in C# is a paradigm that organizes code into reusable, modular components. This tutorial walks through the four pillars — encapsulation, inheritance, abstraction, and polymorphism — showing how to build scalable, maintainable applications using classes, interfaces, and method overrides.

🧪 Try It Yourself

// OOPDemo.cs
using System;

class Animal {
  public virtual void Speak() {
    Console.WriteLine("Animal speaks");
  }
}

class Dog : Animal {
  public override void Speak() {
    Console.WriteLine("Dog barks");
  }
}

class Program {
  static void Main() {
    Animal myAnimal = new Dog(); // Polymorphism
    myAnimal.Speak();            // Output: Dog barks
  }
}
