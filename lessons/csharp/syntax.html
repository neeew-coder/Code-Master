🔍 Explanation
  • using System; imports the System namespace.
  • namespace groups related classes.
  • class Program defines the class.
  • Main() is the entry point.
  • // starts a comment.
  • Statements end with ;.

🧠 Analogy: Think of writing C# code like following a recipe:

  • using System; is like gathering your ingredients.
  • namespace is your kitchen — it organizes where things happen.
  • class Program is the recipe itself — a set of instructions grouped together.
  • Main() is the starting step — the first action you take.
  • Console.WriteLine() is like plating the dish — it shows the result.
  • Semicolons ; are like punctuation in a cookbook — they separate steps clearly.

📚 More Examples

Variable Declaration:
int age = 25;
Console.WriteLine("Age: " + age);

Conditional Block:
if (isDark) { Console.WriteLine("Dark mode"); } else { Console.WriteLine("Light mode"); }

Loop:
for (int i = 0; i < 3; i++) { Console.WriteLine(i); }

💡 Tips: C# is case-sensitive. MyClass and myclass are different. Use indentation and comments to make your code readable.

🧠 Lesson Summary

C# syntax is clean, structured, and highly readable — built around classes, methods, and strong typing.
