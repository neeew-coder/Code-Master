Variables & Types

C# uses strongly typed variables. You must declare the type of data a variable will hold, such as int, string, or bool.

🎯 Learning Objectives

  • Declare variables with types
  • Use basic data types and assign values

🧩 Key Concepts

  • Type safety and initialization
  • String concatenation and numeric operations

🛠️ Real-world Application

Variables are used to store user input, perform calculations, and manage logic in applications.

✅ Expected Outcome

You’ll be able to declare and use variables with appropriate types in C#. 

string name = "Mark";
int age = 21;
Console.WriteLine($"{name} is {age} years old.");
