📦 C# Variables

Variables are containers for storing data values. In C#, every variable has a type that determines what kind of data it can hold. You must declare the type before using the variable.

🔍 Explanation
  • int stores whole numbers.
  • string stores text.
  • double stores decimal numbers.
  • bool stores true/false values.
  • Use = to assign a value to a variable.

📦 Analogy: A variable is like a labeled box. You decide what type of item goes inside (e.g., numbers, text), give the box a name, and then store something in it. Later, you can open the box and use what’s inside.

📚 More Examples

Separate Declaration and Assignment:
int score;
score = 100;

Multiple Variables:
int x = 1, y = 2, z = 3;

💡 Tips: Use meaningful variable names like totalPrice instead of vague ones like tp. Avoid using reserved keywords.
