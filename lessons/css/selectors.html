CSS Selectors

Selectors determine which elements get styled. By using element names, class names, IDs, and grouping techniques, you can control exactly what gets affected by your CSS declarations.

📘 What You’ll Learn

  • How to use element, class, ID, and group selectors
  • How to apply styles to multiple elements efficiently
  • How to distinguish between reusable and unique styling
  • How selector specificity affects styling behavior

🔹 Selector Types

  • Element selector — targets by tag name (p, h1)
  • Class selector — targets by class name (.menu)
  • ID selector — targets by unique ID (#header)
  • Group selector — targets multiple elements (h1, p)

📄 Syntax Examples

p {
  color: navy;
}

.menu {
  background-color: lightgray;
  padding: 10px;
}

#header {
  font-size: 32px;
  color: darkblue;
}

h1, p {
  font-family: Arial;
}

🧠 Selector Tips

  • Use classes for reusable styling across multiple elements
  • Use IDs for unique elements — avoid reusing the same ID
  • Group selectors with commas to apply shared styles
  • Selectors can also target states like :hover, or nested structures — covered in later modules

🧠 Lesson Summary

  • p targets all paragraph elements
  • .menu targets any element with class="menu"
  • #header targets the element with id="header"
  • h1, p applies shared styles to both elements
  • Use classes for reusable styling, IDs for unique elements
