CSS Pseudo-Elements

Pseudo-elements style specific parts of an element — like the first letter, first line, or allow injected content before/after without modifying HTML. They’re written with double colons (::).

🎯 Common Pseudo-Elements

SelectorEffect
::beforeInserts content before the element
::afterInserts content after the element
::first-letterTargets the first letter
::first-lineStyles the first line of a paragraph
::selectionStyles highlighted text
::placeholderStyles input placeholder text

🧪 Syntax Examples

p::first-letter {
  font-size: 2em;
  color: blue;
}

p::before {
  content: "★ ";
  color: steelblue;
}

p::after {
  content: " ✔";
  color: green;
}

input::placeholder {
  color: darkblue;
  font-style: italic;
}

👁️ Visual Preview

This paragraph has styled first letter, injected symbol before, and checkmark after.

🧠 Pseudo-Element Tips

  • Use content: for ::before and ::after — must be set to render
  • ::selection styles browser highlight color (cross-browser caution)
  • ::placeholder can differ in syntax between browsers — use vendor prefixes if needed
  • Double colon :: is standard, but single colon : still works for older browsers
