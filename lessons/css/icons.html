CSS Icons

Icons add visual meaning to interfaces — from navigation to feedback to branding. You can use font-based icons like Font Awesome, inline SVGs, or emoji. Icons scale well and style like text, making them perfect for CSS-based design.

🔤 Font-Based Icons

Libraries like Font Awesome provide thousands of vector icons via CSS classes. First, include the library: 

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

<i class="fas fa-home"></i>
<i class="fas fa-user"></i>
<i class="fas fa-envelope"></i>

📁 SVG Icons

SVGs are scalable and customizable with CSS. You can insert inline or use <img> tags: 

<svg width="24" height="24" viewBox="0 0 24 24" fill="blue">
  <path d="M10 20v-6h4v6h5v-8h3L12 3 2 12h3v8z"/>
</svg>

Home icon using inline SVG

🔸 Styling Icons with CSS

.fas {
  color: steelblue;
  font-size: 32px;
  padding: 10px;
}

Icons behave like text — you can style with color, spacing, and hover effects.

🧠 Icon Tips

  • Font icons require external CSS links — make sure they load before use
  • SVGs offer sharper rendering and can be styled inline
  • Always add accessible aria-label or alt text for usability

🧠 Lesson Summary

  • <i> tags with Font Awesome classes render scalable icons
  • svg icons offer sharper rendering and inline styling
  • Icons can be styled like text: color, size, spacing, hover effects
  • Always include aria-label or alt text for accessibility
