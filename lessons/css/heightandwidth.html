CSS Height and Width

The height and width properties control how tall and wide an element is. They accept fixed units (px), scalable units (%), and viewport-based values for fluid layouts.

📏 Fixed Dimensions

div {
  width: 300px;
  height: 150px;
}
300px × 150px Box

📐 Percentage-Based Sizing

div {
  width: 50%;
}

This sets the width relative to its parent container.

50% Width

📦 Min and Max Values

div {
  min-width: 200px;
  max-width: 600px;
  min-height: 100px;
  max-height: 400px;
}

Used to keep elements flexible but bounded — helpful in responsive design.

🌐 Viewport Units

width: 100vw; /* full browser width */
height: 100vh; /* full browser height */
  • vw = viewport width
  • vh = viewport height

🧠 Sizing Tips

  • Use max-width to prevent layout stretching on large screens
  • Use min-height to enforce vertical space for containers
  • Consider box-sizing: border-box to include padding and border in total size
