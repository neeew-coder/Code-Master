CSS inline-block

The inline-block value lets elements flow inline — side by side — while allowing you to control width, height, and margin like block elements. It's perfect for menus, buttons, badges, and small layout components.

📄 Syntax

.element {
  display: inline-block;
  width: 120px;
  height: 60px;
  padding: 10px;
  background-color: lightblue;
}

🧩 Common Use Cases

  • Horizontal nav menus with padding
  • Button rows and badge groups
  • Card components that size independently
  • Layout chunks where Flexbox isn’t needed

🧠 Spacing Quirks

By default, inline-block elements have whitespace between them (from HTML source). Use:

  • font-size: 0; on parent (then reset for children)
  • HTML comments or tighter markup
  • Flexbox for more precise control

🧠 Lesson Summary

  • inline flows horizontally but ignores width/height
  • block stacks vertically and respects box styling
  • inline-block flows inline but allows full box styling
  • Whitespace between inline-block elements can cause layout gaps
