How to Add CSS

There are three main ways to apply CSS to HTML: inline, internal, and external. Each method controls styling, but their use cases differ based on scale and maintainability.

📘 What You’ll Learn

🧵 1. Inline CSS

Style applied directly to the HTML element using the style attribute. 

<p style="color: blue; font-size: 18px;">Styled with inline CSS</p>

Styled with inline CSS

📦 2. Internal CSS

Style rules placed inside a <style> tag in the <head> section. 

<head>
  <style>
    p {
      color: darkblue;
      font-size: 18px;
    }
  </style>
</head>

<body>
  <p>This is styled with internal CSS</p>
</body>

This is styled with internal CSS

🗂️ 3. External CSS

Styles written in a separate .css file and linked with <link> in the HTML <head>. 

<head>
  <link rel="stylesheet" href="style.css">
</head>

/* style.css */
p {
  color: steelblue;
  font-size: 18px;
}

This is styled using an external stylesheet

🧠 Comparison Table

MethodUse CaseProsCons
Inline Quick one-off styling Easy to apply Hard to maintain
Internal Single-page projects Centralized styling Not reusable across pages
External Multi-page websites Efficient and reusable Requires file management

🧠 Lesson Summary

  • Inline CSS is quick but hard to maintain
  • Internal CSS centralizes styles for single-page projects
  • External CSS is best for scalable, multi-page websites
  • All three methods affect specificity and override behavior
