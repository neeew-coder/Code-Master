CSS Dropdown Menu

Dropdowns are menus that reveal more options on hover or click. This module demonstrates a classic hover-based dropdown using only CSS — no JavaScript needed for interaction.

🎥 Video Tutorial

Video by Bro Code – Watch on YouTube
CSS Dropdown Menu Tutorial covers nested list structure, hover triggers, positioning, and visibility toggling — with clean visuals and practical examples.

📘 What You’ll Learn

  • How to structure dropdowns using nested <ul> and <li>
  • How to show/hide submenus using :hover and display
  • How to position dropdowns using absolute and relative
  • How to layer dropdowns using z-index

📄 Markup Structure

<nav>
  <ul class="menu">
    <li><a href="#">Home</a></li>
    <li class="dropdown">
      <a href="#">Services</a>
      <ul class="submenu">
        <li><a href="#">Web Design</a></li>
        <li><a href="#">SEO</a></li>
        <li><a href="#">Analytics</a></li>
      </ul>
    </li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>

🎨 Styling Logic

.menu {
  list-style: none;
  display: flex;
  background: #1e3a8a;
  padding: 0;
}

.menu li {
  position: relative;
}

.menu a {
  display: block;
  padding: 14px 20px;
  color: white;
  text-decoration: none;
}

.submenu {
  display: none;
  position: absolute;
  top: 100%;
  left: 0;
  background: #3b82f6;
  list-style: none;
  padding: 0;
  min-width: 160px;
}

.submenu li a {
  padding: 10px;
  color: white;
  display: block;
}

.dropdown:hover .submenu {
  display: block;
}

🧠 Dropdown Tips

  • Use position: relative on parent and absolute on submenu
  • :hover controls visibility without JS
  • For accessibility and mobile, dropdowns often use JS toggles instead
  • Add transitions for smoother reveal

🧠 Lesson Summary

  • Wrap nav items in <ul> and nest submenus inside <li>
  • Use position: relative on parent and absolute on submenu
  • Toggle submenu visibility with :hover and display: block
  • Use z-index to ensure dropdowns appear above other content
