CSS Links

Links are crucial for navigation and interaction. CSS lets you style normal, visited, hover, active, and focus states — making your site feel polished and intuitive.

🔗 Link States

  • a:link — unvisited link
  • a:visited — visited link
  • a:hover — mouse over link
  • a:active — clicked link

📄 Styling Example

a {
  text-decoration: none;
  color: blue;
}

a:hover {
  color: darkblue;
  text-decoration: underline;
}

🧠 Link Styling Tips

  • Use :hover and :focus to improve accessibility
  • Remove text-decoration to hide underline if desired
  • Group state styles together: a, a:visited, a:hover, a:active
  • Use classes for consistent navigation styling
