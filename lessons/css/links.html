CSS Links

Links are crucial for navigation and interaction. CSS lets you style normal, visited, hover, active, and focus states — making your site feel polished and intuitive.

📘 What You’ll Learn

  • How to style links using color and text-decoration
  • How to use pseudo-classes like :hover, :visited, :active, and :link
  • How to remove underlines and add hover effects
  • How to customize visited link behavior

🔗 Link States

  • a:link — unvisited link
  • a:visited — visited link
  • a:hover — mouse over link
  • a:active — clicked link

📄 Styling Example

a {
  text-decoration: none;
  color: blue;
}

a:hover {
  color: darkblue;
  text-decoration: underline;
}

🧠 Link Styling Tips

  • Use :hover and :focus to improve accessibility
  • Remove text-decoration to hide underline if desired
  • Group state styles together: a, a:visited, a:hover, a:active
  • Use classes for consistent navigation styling

🧠 Lesson Summary

  • :link targets unvisited links
  • :visited styles links after they’ve been clicked
  • :hover adds interactivity on mouseover
  • :active applies during click
  • text-decoration: none removes underlines
