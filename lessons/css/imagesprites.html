CSS Image Sprites

Image sprites combine multiple graphics into a single file — like icons, buttons, or state indicators — then use background-position to reveal only the needed section. It improves performance by minimizing image requests.

🧩 Sprite Anatomy

.sprite {
  background-image: url('sprite.png');
  background-repeat: no-repeat;
  display: inline-block;
  width: 32px;
  height: 32px;
}

🎯 Positioning Each Icon

.icon-home {
  background-position: 0 0;
}

.icon-user {
  background-position: -32px 0;
}

.icon-email {
  background-position: -64px 0;
}

🔍 Why Use Sprites?

  • Reduce number of image requests = faster page loads
  • Precise control over visual positioning
  • Ideal for icon sets, button states, thumbnails
  • Easy to update multiple icons at once — one file

🧠 Sprite Tips

  • Ensure sprite file is evenly spaced and consistently sized
  • Use image editors to prep exact coordinates
  • Use display: inline-block and set width/height on each icon
  • Combine sprites with :hover for active states

🧠 Lesson Summary

  • Combine multiple icons into one image file
  • Use background-position to shift the visible area
  • Set width and height to match the icon size
  • Use hover states to shift position and show alternate icons
