CSS Borders

Borders add outlines around elements — defining shape, structure, and separation. You can control their width, style, and color to match your design tone.

🧱 Border Syntax

border: 2px solid blue;
  • width — thickness of border (e.g. 2px)
  • style — line type (e.g. solid, dashed, dotted)
  • color — border color (e.g. blue, #333)

🎨 Border Styles

Solid Border
Dashed Border
Dotted Border

📏 Individual Sides

border-top: 4px solid navy;
border-right: 2px dashed lightblue;
border-bottom: 3px dotted steelblue;
border-left: 2px solid skyblue;
Borders applied individually on all four sides.

🧃 Rounded Borders

border: 2px solid blue;
border-radius: 10px;
Rounded borders add softness to box edges.

🧠 Border Properties

PropertyPurpose
borderSets all border sides at once
border-top / bottom / left / rightTargets individual sides
border-styleDefines line type
border-widthControls thickness
border-colorSets border color
border-radiusRounds border corners
