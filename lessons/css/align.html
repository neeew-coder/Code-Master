CSS Text Alignment

Text alignment positions content horizontally within its container. CSS provides options to align left, center, right, or justify — each affecting readability and layout tone.

  • How to center content both horizontally and vertically

📄 Common Alignments

text-align: left;
text-align: center;
text-align: right;
text-align: justify;

Left-aligned text flows naturally for most languages.

text-align: left

Right-aligned text can be useful for captions or numbers.

text-align: right

Center alignment is often used for headings or standalone phrases.

text-align: center

📏 Justified Alignment

p {
  text-align: justify;
}

Justify spreads text so that lines have equal width, useful in columns and printed layouts — but may cause uneven word spacing on web.

📐 Alignment Scope

  • Works on block-level elements (e.g. <p>, <div>, <h1>)
  • Inline elements (e.g. <span>) cannot be aligned with text-align

🧠 Alignment Tips

  • Center text sparingly — ideal for short headlines or callouts
  • Left alignment is default and works best for long paragraphs
  • Use text-align: justify; in print-style columns — carefully
  • Wrap inline content in block containers to apply alignment

🧠 Lesson Summary

  • text-align: center centers inline content in block containers
  • vertical-align aligns inline elements relative to their baseline
  • justify-content controls horizontal alignment in Flexbox
  • align-items controls vertical alignment in Flexbox
