ğŸ§  Counter Tips

  • Use counter-reset to start or restart numbering
  • counter-increment adds one (or a custom value)
  • content: counter(name) or counters(name, separator) injects number
  • Counters work best with pseudo-elements like ::before

ğŸ§  Lesson Summary

  • Use counter-reset to start a counter at zero
  • Apply counter-increment to elements that should increase the count
  • Use content in ::before or ::after to display the value
  • Nest counters for hierarchical numbering (e.g., 1.1, 1.2)
