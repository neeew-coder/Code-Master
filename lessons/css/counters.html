CSS Counters

CSS counters allow you to create and manage automatic numbering — useful for headings, steps, list items, or dynamic labels. You define, increment, and inject them with content.

🔢 Basic Setup

section {
  counter-reset: section-counter;
}

h2::before {
  counter-increment: section-counter;
  content: "Section " counter(section-counter) ": ";
}

This adds a number before each <h2> inside the section — auto-incrementing for each one.

📂 Nested Counters

ol {
  counter-reset: item;
}

li::before {
  counter-increment: item;
  content: counters(item, ".") " ";
}

🧠 Counter Tips

  • Use counter-reset to start or restart numbering
  • counter-increment adds one (or a custom value)
  • content: counter(name) or counters(name, separator) injects number
  • Counters work best with pseudo-elements like ::before
