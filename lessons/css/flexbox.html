CSS Flexbox

Flexbox (Flexible Box Layout) is a one-dimensional layout system — perfect for aligning elements horizontally or vertically with spacing control, dynamic sizing, and responsive flow.

📦 Flex Container Basics

.flex-parent {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
}
  • flex-direction: row, row-reverse, column, column-reverse
  • justify-content: main-axis spacing (start → end)
  • align-items: cross-axis alignment (top ↕ bottom)

📐 Child Item Control

.flex-child {
  flex-grow: 1;
  flex-shrink: 0;
  flex-basis: 100px;
}
  • flex-grow: how much item expands (relative)
  • flex-shrink: how much item shrinks when needed
  • flex-basis: initial size before growing/shrinking

🧩 Common Layout Patterns

- Horizontal nav bar - Sidebar + content stretch - Equal width cards - Footer pinning + sticky headers - Button group alignment

🪛 Flexbox Debug Tips

  • Use browser dev tools to inspect flex containers
  • Try gap instead of margin spacing
  • Remember: child items won’t align unless parent has display: flex
  • flex-wrap: use for row wrapping behavior
