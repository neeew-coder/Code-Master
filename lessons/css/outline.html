CSS Outline

Outlines are lines drawn around elements outside the border — often used to highlight focus or provide emphasis without affecting layout spacing.

📄 Basic Syntax

outline: 2px solid blue;
  • outline-width — thickness of the outline
  • outline-style — line type: solid, dotted, dashed
  • outline-color — color of the outline

👁️ Visual Preview

🔍 Outline vs Border

/* Outline does NOT affect layout */
outline: 2px dashed navy;

/* Border affects element size and spacing */
border: 2px dashed navy;

✨ Outline Offset

You can shift the outline away from the element using outline-offset: 

outline: 2px solid blue;
outline-offset: 4px;

🧠 Outline Tips

  • Use outlines for form field focus or keyboard navigation feedback
  • They don’t take space — unlike borders
  • Helpful for debugging without altering layout
