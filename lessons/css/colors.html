🧠 Lesson Summary

  • Named colors are quick and readable (e.g. red, lightblue)
  • HEX codes use six-digit values for precise web-safe colors
  • RGB defines colors by red, green, and blue intensity (0–255)
  • HSL uses hue, saturation, and lightness — ideal for theme adjustments
  • All formats can be used for color, background-color, border-color, and more
