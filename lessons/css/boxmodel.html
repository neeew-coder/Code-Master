CSS Box Model

Every HTML element is rendered as a rectangular box. The CSS Box Model defines how that box behaves and how space is distributed around and inside it — essential for layout control.

📦 Box Model Layers

  • Content — text, image, or media inside the element
  • Padding — space between content and border
  • Border — outline surrounding padding and content
  • Margin — space outside the border, separating elements

📄 Default Behavior

By default, width and height only apply to the content box. Padding and border add extra space outside that width — unless you change it using box-sizing. 

/* Standard model */
box-sizing: content-box;

/* Better sizing */
box-sizing: border-box;

🧠 Layout Tips

  • Use box-sizing: border-box to include padding & border inside total size
  • Visualize spacing during debugging with DevTools
  • Balance margin for spacing between elements, and padding for internal breathing room
