CSS Backgrounds

The background property group lets you style the backdrop of HTML elements. You can control color, images, positioning, repetition, and sizing for rich visual layering.

🎨 Background Color

div {
  background-color: lightblue;
}
This box has a light blue background.

🖼️ Background Image

body {
  background-image: url('stars.jpg');
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
}

Tip: Use high-resolution images and compress them for performance.

🔁 Repeating Backgrounds

div {
  background-image: url('pattern.png');
  background-repeat: repeat;
}
Repeated pattern background (placeholder image).

📍 Background Position & Size

background-position: top right;
background-size: contain;
  • cover — scales to fill the container
  • contain — scales to fit without cutting
  • no-repeat — disables image tiling

🧠 Common Background Properties

PropertyDescription
