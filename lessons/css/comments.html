CSS Comments

Comments in CSS are used to explain code, mark sections, or temporarily disable styling. They do not affect how the page looks and are ignored by browsers.

💬 Syntax

CSS comments use /* ... */. Anything inside is skipped during rendering. 

/* This is a comment */

p {
  font-size: 18px; /* Font size for paragraph */
}

/* h1 {
  text-align: center;
} */

👁️ Visual Preview

This paragraph is styled — the comment had no effect.

This heading isn't centered because the style is commented out.

🧠 Use Cases

  • Explain what each rule does
  • Disable styling temporarily during testing
  • Mark section breaks or notes for collaborators
