CSS Comments

Comments in CSS are used to explain code, mark sections, or temporarily disable styling. They do not affect how the page looks and are ignored by browsers.

ğŸ§  Lesson Summary

  • CSS comments use /* ... */ and are ignored by browsers
  • They help explain code, organize sections, and disable styles temporarily
  • Comments can appear above rules, inline with declarations, or wrap entire blocks
  • Commenting out a rule removes its visual effect without deleting the code
  • Strategic commenting improves collaboration, debugging, and long-term maintainability
