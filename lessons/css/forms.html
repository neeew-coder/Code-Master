CSS Form Styling

Forms collect user data — making them essential for interaction. CSS lets you style inputs, buttons, labels, and validation states to ensure a clean, accessible, and usable interface.

📋 Sample Form Markup

<form>
  <label>Name:</label>
  <input type="text" placeholder="Enter name" />

  <label>Email:</label>
  <input type="email" required />

  <label>Message:</label>
  <textarea></textarea>

  <button type="submit">Send</button>
</form>

🎨 Form Styling Basics

form {
  display: flex;
  flex-direction: column;
  gap: 12px;
  max-width: 400px;
  margin: 0 auto;
}

input, textarea {
  padding: 10px;
  border: 1px solid #ccc;
  border-radius: 6px;
}

input:focus, textarea:focus {
  outline: none;
  border-color: steelblue;
  box-shadow: 0 0 0 2px #dbeafe;
}

button {
  background: steelblue;
  color: white;
  border: none;
  padding: 10px 16px;
  border-radius: 6px;
  cursor: pointer;
}

button:hover {
  background: #2563eb;
}

🧠 Form Styling Tips

  • Use :focus for strong keyboard navigation feedback
  • Style placeholder and required fields for clarity
  • Group with flex or grid for alignment
  • Add spacing between form elements using gap or margin
