CSS Math Functions

Math functions let you calculate CSS values directly — combining units, enforcing limits, and defining fluid responsiveness. They work inside properties like `width`, `margin`, `font-size`, and more.

🧮 `calc()`

.element {
  width: calc(100% - 40px);
}

Combines percentage and pixel units — great for padding-aware layouts, sidebars, or full-width sections minus fixed offsets.

📏 `min()` and `max()`

.box {
  width: min(600px, 100%);
  font-size: max(16px, 2vw);
}

`min()` chooses the smallest value. `max()` selects the largest — perfect for capping widths and scaling type.

🔧 `clamp()`

.title {
  font-size: clamp(1rem, 2vw, 2.5rem);
}

`clamp()` sets a flexible value with a **minimum**, **preferred**, and **maximum** — ideal for responsive typography.

🧠 Function Tips

  • Always add spaces inside `calc()` operators: calc(100% - 40px)
  • `clamp()` handles scaling and limits in one declaration
  • `min()` and `max()` help enforce layout bounds
  • Use for fluid type, adaptive padding, auto-scaling grids
