CSS Attribute Selectors

Attribute selectors let you apply styles to elements based on their attributes and values — such as type, href, or data-*. This improves specificity and enables dynamic targeting.

📘 What You’ll Learn

  • How to style elements with specific attributes
  • How to use [attr="value"] for exact matches
  • How to use [attr*="value"] for partial matches
  • How to use [attr^="value"] and [attr$="value"] for prefix/suffix targeting

📄 Syntax Variants

/* Exact match */
input[type="text"] {
  border-color: blue;
}

/* Starts with */
a[href^="https"] {
  color: green;
}

/* Ends with */
img[src$=".png"] {
  border-radius: 8px;
}

/* Contains */
button[title*="delete"] {
  background-color: red;
}

🧪 Sample Markup

🧠 Attribute Selector Tips

  • [attr] targets any element with that attribute
  • [attr="value"] matches exact value
  • ^=, $=, and *= match start, end, or partial values
  • Combine with tags or classes for more precision: input[type="email"]

🧠 Lesson Summary

  • [href] targets all elements with an href attribute
  • [target="_blank"] styles links that open in a new tab
  • [type="email"] targets email input fields
  • [alt*="logo"] styles images with “logo” in their alt text

🧪 Try It Yourself

Create a file named css-attributes-demo.html and link it to style.css. Then:

  • Style all external links using [target="_blank"]
  • Highlight email fields with [type="email"]
  • Style images with alt text containing “icon” using [alt*="icon"]
  • Use [href^="https"] to style secure links
