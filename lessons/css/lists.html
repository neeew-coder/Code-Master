CSS Lists

Lists organize content into bullet points or sequences. CSS gives you control over marker styles, positioning, spacing, and even custom numbering logic.

📄 List Types

  • <ul> — unordered list (bullets)
  • <ol> — ordered list (numbers)
  • <li> — each list item
<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>

<ol>
  <li>Step One</li>
  <li>Step Two</li>
</ol>

👁️ Visual Preview

  • Unordered item
  • Another item
  1. First step
  2. Second step

🎯 Styling Markers

ul {
  list-style-type: square;
}

ol {
  list-style-type: upper-roman;
}
  • Square bullet
  • Another item
  1. First
  2. Second

📐 Marker Positioning

li {
  list-style-position: inside;
}

Use inside or outside to control if bullets sit within or outside text block.

✨ Remove Bullets

ul {
  list-style-type: none;
}
  • No bullet item
  • Just clean text

🧠 List Styling Tips

  • Use list-style-type for marker control
  • list-style-position adjusts marker alignment
  • You can add padding/spacing to improve readability
  • Use list-style: none; on nav menus for custom styling
