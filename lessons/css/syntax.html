CSS Syntax

CSS syntax uses selectors to target elements, then applies one or more declarations inside curly braces. Each declaration defines a style property and its value, ending with a semicolon.

🔹 Structure Overview

  • selector — targets the HTML element
  • property — the visual style (e.g. color, font-size)
  • value — the setting for that property
selector {
  property: value;
}

📄 Example: Styling Elements

h1 {
  color: navy;
  font-size: 36px;
  text-align: center;
}

p {
  font-family: Arial, sans-serif;
  font-size: 18px;
  color: #333;
}

📘 Notes on Syntax

  • Multiple declarations inside one block must be separated by semicolons
  • Selectors can be elements (p), classes (.title), or IDs (#header)
  • Whitespace doesn't affect rendering — it's for readability
  • selector targets HTML elements like h1, p, .class, or #id
  • Each declaration uses property: value; syntax
  • Multiple declarations are separated by semicolons inside curly braces
