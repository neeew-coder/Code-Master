CSS Navigation Bar

Navigation bars are horizontal or vertical groups of links that help users move between sections. A clean navbar improves site usability and sets visual tone.

🎥 Video Tutorial

Video by Bro Code – Watch on YouTube
CSS Navigation Bar Tutorial covers horizontal nav layout, hover effects, active link styling, and spacing — with clean visuals and practical examples.

📘 What You’ll Learn

  • How to structure a navigation bar using semantic HTML
  • How to style nav links with display: inline-block
  • How to add hover and active effects
  • How to space and align nav items horizontally

📄 Basic Markup

<nav>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Services</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>

🎨 Styling Logic

nav ul {
  display: flex;
  list-style: none;
  padding: 0;
  background-color: #1e3a8a;
}

nav li {
  margin-right: 20px;
}

nav a {
  display: block;
  color: white;
  text-decoration: none;
  padding: 14px 20px;
}

nav a:hover {
  background-color: #3b82f6;
}

nav a.active {
  background-color: #2563eb;
}

🧠 Navbar Tips

  • Use flex layout for horizontal nav bars
  • :hover highlights each link on interaction
  • Add an .active class for current page indication
  • For mobile nav, use media queries or toggleable menus

🧠 Lesson Summary

  • Use <nav> and <ul> for semantic structure
  • display: inline-block allows horizontal flow with box styling
  • :hover adds interactivity on mouseover
  • .active class highlights the current page
  • Use padding and margin for spacing
