CSS Float and Clear

The float property lets elements move beside others — like text wrapping around an image. But it also removes them from normal document flow, so clear helps reset and restore layout structure.

📘 What You’ll Learn

  • How float: left and float: right affect layout
  • How text wraps around floated elements
  • How to clear floats using clear or clearfix methods
  • How to use floats for image alignment and sidebars

📄 Float Syntax

img {
  float: left;
  margin-right: 20px;
}

aside {
  float: right;
  width: 200px;
}

👁️ Float Preview

This text wraps around the floated image. Floats move elements to one side, letting text flow beside them — a classic newspaper-style layout.

🧼 Clear Syntax

.clearfix::after {
  content: "";
  display: table;
  clear: both;
}

Use clear to stop floating influence and preserve vertical stacking. The clearfix method keeps containers from collapsing when inner floats are used.

📐 Direct Clear Usage

footer {
  clear: both;
}

🧠 Float Tips

  • Floats are great for image-text layouts or sidebars
  • Use clear on the next block element to reset flow
  • Floating elements shrink container height unless cleared
  • Modern layouts prefer Flexbox or Grid — but float is still useful!

🧠 Lesson Summary

  • float removes elements from normal flow and aligns them left or right
  • Text and inline content wrap around floated elements
  • Floats can cause layout collapse — use clear or clearfix to fix
  • Modern layouts prefer Flexbox or Grid, but float is still useful for specific cases
