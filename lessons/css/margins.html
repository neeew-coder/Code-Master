CSS Margins

Margins create space outside an element’s border — separating it from neighbors. You can apply margin to one side, opposite sides, or all sides at once using shorthand.

📄 Syntax

div {
  margin: 20px;              /* all sides */
  margin-left: 10px;         /* left only */
  margin: 10px 20px;         /* vertical | horizontal */
  margin: 10px 15px 20px 25px; /* top | right | bottom | left */
}

🎯 Auto-Centering

Horizontally center a block with auto side margins: 

div {
  width: 300px;
  margin: 0 auto;
}
Centered with margin: 0 auto;

🧠 Margin Notes

  • Margin collapses between stacked elements (p + p) — only the largest margin is applied
  • Negative margins are allowed but can break layout
  • auto helps with alignment, especially in centering blocks

🧠 Lesson Summary

  • margin sets spacing outside the border of an element
  • Shorthand supports 1–4 values: all sides, vertical/horizontal, or top/right/bottom/left
  • Individual properties like margin-top target specific sides
  • margin: 0 auto centers block elements horizontally
  • Margins between stacked elements collapse — only the larger one is applied
