CSS Display and Visibility

CSS controls how elements behave in layout flow. The display and visibility properties define whether elements stack, line up, disappear, or remain part of the structure invisibly.

📘 What You’ll Learn

  • How display affects layout flow and stacking
  • Differences between block, inline, and inline-block
  • How to hide elements using display: none
  • How to use display for responsive layout control

📦 Display Values

  • block — stacks vertically, full-width
  • inline — flows within text, width not enforced
  • inline-block — flows inline but allows width/height
  • none — element is completely removed from layout
div {
  display: block;
}

span {
  display: inline;
}

button {
  display: inline-block;
}

🙈 Visibility Control

Use visibility to hide elements but keep layout intact. 

.hidden-box {
  visibility: hidden;
}

.none-box {

🧠 Display Tips

  • display: none removes the element from the flow
  • visibility: hidden keeps the space reserved
  • inline-block is great for buttons and cards
  • Flex and Grid introduce advanced layout behaviors — covered soon

🧠 Lesson Summary

  • display: block makes elements stack vertically and take full width
  • display: inline flows elements horizontally without width/height control
  • display: inline-block combines inline flow with block styling
  • display: none removes the element from layout and rendering
