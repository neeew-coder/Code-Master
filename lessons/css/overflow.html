CSS Overflow

The overflow property controls how content behaves when it exceeds the bounds of its container. You can hide, scroll, or clip overflow — depending on layout goals.

📘 What You’ll Learn

  • How overflow affects content visibility and scrollability
  • How to use visible, hidden, scroll, and auto
  • How to create scrollable containers
  • How to prevent layout breakage from overflowing content

📄 Overflow Values

  • visible — default; content spills outside
  • hidden — excess content is clipped
  • scroll — always show scrollbars
  • auto — adds scrollbars only if needed
.container {
  width: 200px;
  height: 100px;
  overflow: auto;
}

🧭 Directional Overflow

overflow-x: scroll;
overflow-y: hidden;

Control horizontal and vertical overflow separately. Useful for long code lines, tables, or horizontal galleries.

🧠 Overflow Tips

  • Set height or max-height to trigger overflow behavior
  • Use overflow: scroll; to force scrollbars for predictable layout
  • Combine overflow with box-sizing for better clipping control

🧠 Lesson Summary

  • overflow: visible shows content outside the container
  • overflow: hidden clips content without scrollbars
  • overflow: scroll forces scrollbars even if not needed
  • overflow: auto adds scrollbars only when necessary
  • Use max-height or max-width to trigger overflow behavior
