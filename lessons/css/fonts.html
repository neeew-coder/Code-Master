CSS Fonts

Fonts define the typographic tone of your site. CSS lets you control family, size, style, weight, spacing, and fallback behavior to create readable, expressive text.

📚 Font Families

Use system fonts, web-safe fonts, or web fonts (like Google Fonts). Always define fallback families. 

body {
  font-family: "Arial", sans-serif;
}

h1 {
  font-family: "Georgia", serif;
}

p {
  font-family: "Courier New", monospace;
}

🔠 Font Size

p {
  font-size: 16px;
}
h1 {
  font-size: 2em;
}

Use px, em, rem, % — depending on responsiveness goals.

💪 Font Weight

h1 {
  font-weight: bold;      /* keyword */
}

p {
  font-weight: 300;       /* numeric */
}

🎭 Font Style

p {
  font-style: italic;
}

🧠 Font Notes

  • Use fallback fonts to ensure rendering if the first choice isn’t supported
  • Web fonts require linking via <link> or @import
  • em scales relative to parent size; rem scales to root
