CSS Opacity

The opacity property sets the transparency level of an element — from fully visible (1) to fully invisible (0). It’s perfect for layering effects, faded transitions, and emphasis control.

📄 Syntax

.element {
  opacity: 0.5; /* 50% visible */
}

👁️ Visual Preview

Opacity: 1
Opacity: 0.5
Opacity: 0.2

🧠 Opacity Tips

  • opacity affects entire element — including text and children
  • Use rgba() or hsla() for transparent background only
  • Combine with :hover or transitions for smooth fades
  • Higher opacity = more visibility; lower = more transparent

💡 Targeting Background vs Whole Element

/* Transparent background only */
.element {
  background-color: rgba(70, 130, 180, 0.5);
}
Background only is transparent — text stays solid
