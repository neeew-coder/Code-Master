CSS Units

Units define the size and spacing of elements in CSS — from pixels to percentages, viewport widths to scalable ems. Understanding units helps create precise, flexible, and responsive designs.

🧠 Unit Tips

  • Use rem for consistent spacing across modules
  • em stacks — 1.2em inside 1.2em compounds
  • Use vw and vh for full-screen layouts or responsive wrappers
  • fr units work inside CSS Grid only

🧠 Lesson Summary

  • px is pixel-based and device-dependent
  • em is relative to the parent’s font size
  • rem is relative to the root element’s font size
  • % is relative to the parent’s dimension
  • vw and vh are relative to the viewport width and height
