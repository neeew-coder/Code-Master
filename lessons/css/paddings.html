CSS Padding

Padding creates space inside an element’s border — keeping the content from touching the edges. It improves readability, breathing room, and overall visual comfort.

📄 Syntax

div {
  padding: 20px;                /* all sides */
  padding-left: 10px;           /* left only */
  padding: 10px 20px;           /* vertical | horizontal */
  padding: 10px 15px 20px 25px; /* top | right | bottom | left */
}

🧠 Padding Notes

  • Padding is inside the border — unlike margin
  • Padding affects overall box size unless box-sizing: border-box is set
  • Can be used to add spacing around text, images, buttons, or any content block

🧠 Lesson Summary

  • padding adds space inside the border of an element
  • Shorthand supports 1–4 values: all sides, vertical/horizontal, or top/right/bottom/left
  • Individual properties like padding-top target specific sides
  • Padding affects box size unless box-sizing: border-box is used
  • Useful for spacing text, buttons, images, and layout blocks
