Responsive Design

Responsive design makes websites adapt across screen sizes — from mobile to desktop. With flexible units, breakpoints, and dynamic layouts, CSS helps interfaces remain readable and usable on any device.

📐 Fluid Layout

.container {
  max-width: 100%;
  padding: 1.5rem;
}

.block {
  width: 80%;
  font-size: clamp(1rem, 2vw, 2rem);
}

👁️ Visual Preview

This block resizes fluidly and scales its text.

📱 Media Queries

@media (max-width: 768px) {
  .nav {
    flex-direction: column;
    gap: 12px;
  }

  .block {
    width: 100%;
    font-size: 1rem;
  }
}
