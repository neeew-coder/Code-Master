Responsive Design

Responsive design makes websites adapt across screen sizes — from mobile to desktop. With flexible units, breakpoints, and dynamic layouts, CSS helps interfaces remain readable and usable on any device.

📘 What You’ll Learn

  • How to use @media queries to target screen sizes
  • How to use relative units like %, em, rem, vw, and vh
  • How to build mobile-first layouts that scale up
  • How to test responsiveness using browser dev tools

📐 Fluid Layout

.container {
  max-width: 100%;
  padding: 1.5rem;
}

.block {
  width: 80%;
  font-size: clamp(1rem, 2vw, 2rem);
}

👁️ Visual Preview

This block resizes fluidly and scales its text.

📱 Media Queries

@media (max-width: 768px) {
  .nav {
    flex-direction: column;
    gap: 12px;
  }

  .block {
    width: 100%;
    font-size: 1rem;
  }
}

🧠 Notes

  • Start with mobile-first — target small screens first
  • Use clamp() or vw for scalable typography
  • auto-fill adapts grid items to fit rows as space allows
  • Media queries update layout at defined screen widths

🧠 Lesson Summary

  • @media queries apply styles based on screen width
  • Use max-width and min-width breakpoints for layout shifts
  • Relative units scale with screen or parent size
  • Mobile-first design starts with small screens and adds styles for larger ones
