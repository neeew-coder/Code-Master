🔧 Java Methods & Parameters

A method in Java is a reusable block of code that performs a specific task. Methods can accept parameters (inputs) and optionally return values. This makes your code more flexible, organized, and reusable.

📘 What Are Parameters?

Parameters are like ingredients you pass into a method. They allow the method to work with different data each time it's called. You define them inside the parentheses when declaring the method, and provide actual values (called arguments) when calling it.

📘 Method Syntax

static ReturnType methodName(Type param1, Type param2) {
  // method body
  return value; // if not void
}
