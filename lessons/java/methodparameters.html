🔧 Java Methods & Parameters

A method in Java is a reusable block of code that performs a specific task. Methods can accept parameters (inputs) and optionally return values. This makes your code more flexible, organized, and reusable.

📘 What Are Parameters?

Parameters are like ingredients you pass into a method. They allow the method to work with different data each time it's called. You define them inside the parentheses when declaring the method, and provide actual values (called arguments) when calling it.

📘 Method Syntax

static ReturnType methodName(Type param1, Type param2) {
  // method body
  return value; // if not void
}

🧠 Lesson Summary

Method parameters in Java allow you to pass data into a method for processing. Each parameter has a type and name, and arguments must match these when calling the method. Java uses pass-by-value for primitives and pass-by-reference-like behavior for objects. Understanding how parameters work enables modular design, flexible logic, and clean code reuse.

🧪 Try It Yourself

// ParameterDemo.java
public class ParameterDemo {
  public static void main(String[] args) {
    greet("Mark", 3);
  }

  static void greet(String name, int times) {
    for (int i = 0; i < times; i++) {
      System.out.println("Hello, " + name + "!");
    }
  }
}
