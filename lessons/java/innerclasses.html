🏗️ Java Inner Classes

An inner class is a class defined within another class. It helps logically group classes that are only used in one place, and can access members of the outer class directly.

📘 Types of Inner Classes
  • Member Inner Class: Non-static, tied to an instance of the outer class
  • Static Nested Class: Static, does not require outer class instance
  • Local Inner Class: Defined inside a method
  • Anonymous Inner Class: No name, used for one-time implementation

🧠 Lesson Summary

Inner classes in Java are classes defined within another class. They help logically group code that’s only used in one place, improve encapsulation, and simplify event-driven programming. Non-static inner classes have access to outer class members, while static inner classes do not. Anonymous and local inner classes offer concise ways to implement interfaces or override methods inline.

🧪 Try It Yourself

// InnerClassDemo.java
class Outer {
  int outerValue = 10;

  class Inner {
    void show() {
      System.out.println("Outer value: " + outerValue);
    }
  }

  static class StaticInner {
    void display() {
      System.out.println("Inside static inner class");
    }
  }
}

public class InnerClassDemo {
  public static void main(String[] args) {
    Outer outer = new Outer();
    Outer.Inner inner = outer.new Inner();
    inner.show();

    Outer.StaticInner staticInner = new Outer.StaticInner();
    staticInner.display();
  }
}
