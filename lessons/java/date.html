📅 Java Date and Time

Java provides the java.time package to handle dates and times in a modern, immutable, and thread-safe way. Key classes include LocalDate, LocalTime, and LocalDateTime.

📘 Why Use java.time?
  • Immutable and thread-safe design
  • Clear separation of date, time, and datetime
  • Built-in formatting and parsing
  • Easy arithmetic and comparison
