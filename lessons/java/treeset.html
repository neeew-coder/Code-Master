🌲 Java TreeSet

TreeSet is a Set implementation that stores elements in sorted order. It uses a Red-Black tree under the hood and ensures uniqueness and ordering.

📘 Why Use TreeSet?
  • Automatically sorts elements in natural or custom order
  • Prevents duplicates like all sets
  • Efficient for range queries and ordered iteration
  • Supports methods like first(), last(), higher(), lower()

🧠 Lesson Summary

TreeSet is a Java collection that stores unique elements in sorted order. It uses a Red-Black tree internally to maintain order and supports navigation methods like first(), last(), and subSet(). It’s ideal for range queries, ordered iteration, and scenarios where sorting and uniqueness are both required.

🧪 Try It Yourself

// TreeSetDemo.java
import java.util.TreeSet;

public class TreeSetDemo {
  public static void main(String[] args) {
    TreeSet scores = new TreeSet<>();
    scores.add(85);
    scores.add(92);
    scores.add(75);
    scores.add(85); // Duplicate ignored

    System.out.println("Scores: " + scores);
    System.out.println("Lowest: " + scores.first());
    System.out.println("Highest: " + scores.last());
    System.out.println("Between 80 and 95: " + scores.subSet(80, 95));
  }
}
