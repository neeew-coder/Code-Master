Java Output / Print

You can use the System.out.println() method to output values or print text in Java.

🔤 Double Quotes

Text must be wrapped inside double quotation marks "". If you forget them, an error occurs: 

System.out.println("This sentence will work!");
System.out.println(This sentence will produce an error);

🖨️ The print() Method

The print() method is similar to println(), but it does not insert a new line: 

System.out.print("Hello World! ");
System.out.print("I will print on the same line.");

Note: We added a space after "Hello World!" for better readability.

🔢 Print Numbers

You can also use System.out.println() to print numbers. Unlike text, numbers are not wrapped in quotes: 

System.out.println(3);
System.out.println(358);
System.out.println(50000);

➕ Math Inside println()

You can perform calculations directly inside the println() method: 

System.out.println(3 + 3);
System.out.println(2 * 5);
