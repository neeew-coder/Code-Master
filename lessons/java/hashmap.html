🔑 Java HashMap

HashMap is a Map implementation that stores key-value pairs using hashing. It allows fast access, insertion, and deletion, with no guaranteed order.

📘 Why Use HashMap?
  • Fast lookup and update via keys
  • Keys must be unique; values can repeat
  • Efficient for indexing, caching, and associations
  • Not ordered — use LinkedHashMap for order

🧠 Lesson Summary

HashMap is a Java collection that stores data as key-value pairs with no guaranteed order. It uses hashing to provide fast access, insertion, and deletion. Keys must be unique, and values can be retrieved efficiently using the key. It’s ideal for dictionaries, caches, and lookup tables.

🧪 Try It Yourself

// HashMapDemo.java
import java.util.HashMap;

public class HashMapDemo {
  public static void main(String[] args) {
    HashMap capitals = new HashMap<>();
    capitals.put("Philippines", "Manila");
    capitals.put("Japan", "Tokyo");
    capitals.put("France", "Paris");

    System.out.println("Capital of Japan: " + capitals.get("Japan"));
    capitals.remove("France");

    for (String country : capitals.keySet()) {
      System.out.println(country + " → " + capitals.get(country));
    }
  }
}
