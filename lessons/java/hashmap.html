🔑 Java HashMap

HashMap is a Map implementation that stores key-value pairs using hashing. It allows fast access, insertion, and deletion, with no guaranteed order.

📘 Why Use HashMap?
  • Fast lookup and update via keys
  • Keys must be unique; values can repeat
  • Efficient for indexing, caching, and associations
  • Not ordered — use LinkedHashMap for order
