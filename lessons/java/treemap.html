🌲 Java TreeMap

TreeMap is a Map implementation that stores key-value pairs in sorted order. It uses a Red-Black tree and maintains natural or custom key ordering.

📘 Why Use TreeMap?
  • Automatically sorts keys in ascending order
  • Efficient for range queries and ordered iteration
  • Keys must be comparable or use a custom comparator
  • Useful for ranking, indexing, and ordered maps

🧠 Lesson Summary

TreeMap is a Java collection that stores key-value pairs in ascending key order using a Red-Black tree. It supports efficient navigation and range queries, making it ideal for ordered maps, sorted dictionaries, and interval-based logic. Unlike HashMap, it guarantees key order and supports comparator-based customization.

🧪 Try It Yourself

// TreeMapDemo.java
import java.util.*;

public class TreeMapDemo {
  public static void main(String[] args) {
    TreeMap ranks = new TreeMap<>();
    ranks.put(3, "Bronze");
    ranks.put(1, "Gold");
    ranks.put(2, "Silver");

    System.out.println("Ranks: " + ranks);
    System.out.println("Top rank: " + ranks.firstEntry());
    System.out.println("Lowest rank: " + ranks.lastEntry());
  }
}
