ğŸ§  Lesson Summary

TreeMap is a Java collection that stores key-value pairs in ascending key order using a Red-Black tree. It supports efficient navigation and range queries, making it ideal for ordered maps, sorted dictionaries, and interval-based logic. Unlike HashMap, it guarantees key order and supports comparator-based customization.

// TreeMapDemo.java
import java.util.*;

public class TreeMapDemo {
  public static void main(String[] args) {
    TreeMap ranks = new TreeMap<>();
    ranks.put(3, "Bronze");
    ranks.put(1, "Gold");
    ranks.put(2, "Silver");

    System.out.println("Ranks: " + ranks);
    System.out.println("Top rank: " + ranks.firstEntry());
    System.out.println("Lowest rank: " + ranks.lastEntry());
  }
}
