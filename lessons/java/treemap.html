🌲 Java TreeMap

TreeMap is a Map implementation that stores key-value pairs in sorted order. It uses a Red-Black tree and maintains natural or custom key ordering.

📘 Why Use TreeMap?
  • Automatically sorts keys in ascending order
  • Efficient for range queries and ordered iteration
  • Keys must be comparable or use a custom comparator
  • Useful for ranking, indexing, and ordered maps
