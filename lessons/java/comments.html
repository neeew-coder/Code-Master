Java Comments

Comments can be used to explain Java code and make it more readable. They can also be used to prevent execution when testing alternative code.

🗒️ Single-line Comments

Single-line comments start with two forward slashes //. Any text between // and the end of the line is ignored by Java.

Output will appear here.

📋 Multi-line Comments

Multi-line comments start with /* and end with */. Any text between them is ignored by Java. 

/* The code below will print the words Hello World
to the screen, and it is amazing */
System.out.println("Hello World");

🧠 Which to Use?

It's up to you! Use // for short comments and /* */ for longer explanations or comment blocks.
