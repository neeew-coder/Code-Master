🏗️ Java Constructors

A constructor is a special method used to initialize objects. It runs automatically when an object is created. Constructors can be default, parameterized, or overloaded.

📘 Constructor Basics
  • Same name as class
  • No return type (not even void)
  • Automatically called when using new
  • Can be overloaded to accept different parameters

🧠 Lesson Summary

Constructors in Java are special methods used to initialize new objects. They share the class name and have no return type. A default constructor takes no arguments, while parameterized constructors allow custom initialization. Overloading lets you define multiple constructors with different signatures. Constructors ensure objects start with valid state and support clean, readable instantiation.

🧪 Try It Yourself

// ConstructorDemo.java
class Book {
  String title;
  int pages;

  Book() {
    this.title = "Untitled";
    this.pages = 0;
  }

  Book(String title, int pages) {
    this.title = title;
    this.pages = pages;
  }

  void display() {
    System.out.println("Title: " + title + ", Pages: " + pages);
  }
}

public class ConstructorDemo {
  public static void main(String[] args) {
    Book b1 = new Book();
    Book b2 = new Book("Java Basics", 120);
    b1.display();
    b2.display();
  }
}
