🧬 Java Inheritance

Inheritance allows one class to acquire the properties and behaviors of another. It promotes code reuse and supports polymorphism. In Java, inheritance is implemented using the extends keyword.

📘 What You’ll Learn

  • How inheritance allows a subclass to reuse and extend a superclass
  • How to use the extends keyword to create a class hierarchy
  • How inherited methods and fields are accessed and overridden
  • How constructors behave in inheritance chains
  • How inheritance supports polymorphism and abstraction
📘 Why Use Inheritance?
  • Reuse existing code without rewriting
  • Organize related classes into hierarchies
  • Enable polymorphism (same method name, different behavior)
  • Improve maintainability and scalability

🧠 Analogy: Think of inheritance like a family tree. You inherit traits from your parents (base class), but you can also have your own unique features (subclass).

🧠 Lesson Summary

Inheritance in Java lets one class acquire the properties and behaviors of another. A subclass uses the extends keyword to inherit from a superclass, gaining access to its fields and methods. This promotes code reuse, logical hierarchy, and polymorphic behavior. Overriding allows subclasses to customize inherited methods, while constructors ensure proper initialization across the chain.

🧪 Try It Yourself

// InheritanceDemo.java
class Animal {
  void makeSound() {
    System.out.println("Animal makes sound");
  }
}

class Dog extends Animal {
  void makeSound() {
    System.out.println("Dog barks");
  }
}

public class InheritanceDemo {
  public static void main(String[] args) {
    Animal a = new Dog(); // Polymorphism
    a.makeSound(); // Output: Dog barks
  }
}
