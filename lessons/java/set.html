🔁 Java Set Interface

A Set is a collection that does not allow duplicate elements. It models mathematical sets and is useful for filtering, uniqueness, and membership checks.

📘 Why Use Set?
  • Automatically removes duplicates
  • Efficient membership testing with contains()
  • Useful for tags, filters, and unique values
  • Common implementations: HashSet, LinkedHashSet, TreeSet
