🔁 Java Set Interface

A Set is a collection that does not allow duplicate elements. It models mathematical sets and is useful for filtering, uniqueness, and membership checks.

🧠 Lesson Summary

Java’s Set interface defines a collection that cannot contain duplicate elements. HashSet is the most commonly used implementation, offering constant-time performance for basic operations. LinkedHashSet maintains insertion order, while TreeSet keeps elements sorted. Sets are ideal for membership testing, uniqueness enforcement, and mathematical set operations.

🧪 Try It Yourself

// SetDemo.java
import java.util.*;

public class SetDemo {
  public static void main(String[] args) {
    Set tools = new HashSet<>();
    tools.add("Java");
    tools.add("Python");
    tools.add("Java"); // Duplicate ignored

    System.out.println("Tools: " + tools);
    System.out.println("Contains Python? " + tools.contains("Python"));

    tools.remove("Python");
    System.out.println("After removal: " + tools);
  }
}
