🧠 Lesson Summary

Java operators are symbols that perform operations on variables and values. Arithmetic operators handle math, relational operators compare values, and logical operators control decision-making. Assignment operators update variables efficiently, and understanding precedence ensures expressions evaluate correctly. Mastering operators is essential for writing clean, logical, and bug-free Java code.

🧪 Try It Yourself

// OperatorDemo.java
public class OperatorDemo {
  public static void main(String[] args) {
    int a = 10, b = 5;
    System.out.println("Add: " + (a + b));
    System.out.println("Equal: " + (a == b));
    System.out.println("Logical AND: " + (a > 0 && b > 0));
    a += 3;
    System.out.println("Updated a: " + a);
  }
}
