To check if Java is installed on your PC, open cmd.exe and type: 


C:\Users\Your Name>java -version

If installed, you'll see something like: 


java version "22.0.0" 2024-08-21 LTS  
Java(TM) SE Runtime Environment 22.9 (build 22.0.0+13-LTS)  
Java HotSpot(TM) 64-Bit Server VM 22.9 (build 22.0.0+13-LTS, mixed mode)

If not, download Java from oracle.com.

üìÇ Writing and Running Java Locally

Save the following code in a file called Main.java using any text editor: 


public class Main {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}

Then open cmd.exe, navigate to the file location, and type: 


C:\Users\Your Name>javac Main.java  
C:\Users\Your Name>java Main

The output should be: 

Hello World

üìö Explanation

  • public class Main defines the class named Main.
  • public static void main(String[] args) is the entry point of every Java application.
  • System.out.println("Hello World"); prints text to the console.
