ğŸ“‚ Writing and Running Java Locally

Save the following code in a file called Main.java using any text editor: 


public class Main {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}

Then open cmd.exe, navigate to the file location, and type: 


C:\Users\Your Name>javac Main.java  
C:\Users\Your Name>java Main

The output should be: 

Hello World
