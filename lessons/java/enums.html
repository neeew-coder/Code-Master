🎯 Java Enums

An enum (short for "enumeration") is a special Java type used to define a collection of constants. Enums are type-safe and can include fields, methods, and constructors.

📘 Why Use Enums?
  • Group related constants under one type
  • Improve type safety and readability
  • Support custom behavior per constant
  • Work seamlessly with switch statements
