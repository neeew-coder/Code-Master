🎯 Java Enums

An enum (short for "enumeration") is a special Java type used to define a collection of constants. Enums are type-safe and can include fields, methods, and constructors.

  • Improve type safety and readability
  • Support custom behavior per constant
  • Work seamlessly with switch statements

🧠 Lesson Summary

Enums in Java are special classes that represent a fixed set of constants. They’re ideal for modeling states, categories, or options — like days of the week or user roles. Unlike primitive constants, enums are type-safe and can include fields, methods, and constructors. They simplify control flow and reduce bugs by enforcing valid values.

🧪 Try It Yourself

// EnumDemo.java
enum Level {
  LOW, MEDIUM, HIGH
}

public class EnumDemo {
  public static void main(String[] args) {
    Level l = Level.HIGH;

    switch (l) {
      case LOW -> System.out.println("Low level");
      case MEDIUM -> System.out.println("Medium level");
      case HIGH -> System.out.println("High level");
    }
  }
}
