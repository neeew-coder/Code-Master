🧭 Java this Keyword

The this keyword refers to the current object. It’s used to resolve naming conflicts, pass the current object as a parameter, or chain constructors. It helps clarify which variable or method belongs to the object itself.

📘 When to Use this
  • Resolve shadowing: When method parameters have the same name as fields.
  • Pass current object: To other methods or constructors.
  • Chain constructors: Using this() to call another constructor.
