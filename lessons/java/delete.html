🔍 Explanation

File.delete() returns true if the file was successfully deleted. Always check file.exists() first to avoid unnecessary operations.

🧠 Analogy: Deleting a file is like throwing away a document. You check if it’s there, then toss it — but sometimes the trash bin might be locked!

🧠 Lesson Summary

Java’s File class provides a simple method delete() to remove files from the filesystem. Before deletion, it’s best practice to check file existence and confirm success afterward. This helps avoid silent failures and ensures proper resource cleanup. Deletion is irreversible, so exception handling and user feedback are essential.

🧪 Try It Yourself

// DeleteFileDemo.java
import java.io.File;

public class DeleteFileDemo {
  public static void main(String[] args) {
    File file = new File("old_data.txt");

    if (file.exists()) {
      if (file.delete()) {
        System.out.println("Deleted file: " + file.getName());
      } else {
        System.out.println("Failed to delete file.");
      }
    } else {
      System.out.println("File does not exist.");
    }
  }
}
