🧠 Lesson Summary

Java’s data structures are part of the java.util package and include powerful tools for organizing and accessing data. Lists like ArrayList and LinkedList support ordered collections, while sets like HashSet enforce uniqueness. Maps like HashMap enable fast key-based retrieval. Understanding their trade-offs helps developers write efficient, scalable code.

🧪 Try It Yourself

// DataStructureDemo.java
import java.util.*;

public class DataStructureDemo {
  public static void main(String[] args) {
    List list = new ArrayList<>();
    list.add("Apple");
    list.add("Banana");

    Set set = new HashSet<>(list);
    set.add("Apple"); // Duplicate ignored

    Map map = new HashMap<>();
    map.put("Apple", 2);
    map.put("Banana", 5);

    System.out.println("List: " + list);
    System.out.println("Set: " + set);
    System.out.println("Map: " + map);
  }
}
