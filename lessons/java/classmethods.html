🧩 Java Class Methods

Methods define the actions that a class or object can perform. They can be instance methods (called on objects) or static methods (called on the class itself). Access modifiers like public and private control visibility.

📘 Types of Methods
  • Instance Methods: Operate on object-specific data. Called using an object.
  • Static Methods: Operate independently of objects. Called using the class name.
  • Access Modifiers: Control method visibility (public, private, protected).
