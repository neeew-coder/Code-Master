Class methods in Java define the behavior of objects. Instance methods use object fields and require an object to be called, while static methods belong to the class itself and can be called without instantiation. // MethodDemo.java
class Calculator {
  int add(int a, int b) {
    return a + b;
  }

  static double square(double x) {
    return x * x;
  }
}

public class MethodDemo {
  public static void main(String[] args) {
    Calculator calc = new Calculator();
    System.out.println("Sum: " + calc.add(3, 4));
    System.out.println("Square: " + Calculator.square(5.5));
  }
}
