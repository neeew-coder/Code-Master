🧺 Java Collections Framework

Java Collections provide reusable data structures and algorithms. They make it easy to store, retrieve, and manipulate groups of objects.

📘 What Are Collections?
  • Collections are part of the java.util package
  • They include interfaces like List, Set, Map, and Queue
  • Each interface has multiple implementations optimized for different use cases
