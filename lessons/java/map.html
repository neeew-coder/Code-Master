🗺 Java Map Interface

A Map stores key-value pairs. Each key maps to a single value, and keys must be unique. Maps are ideal for fast lookups, associations, and indexing.

  • Associates keys with values
  • Fast access via get(key)
  • Keys are unique; values can repeat
  • Common types: HashMap, TreeMap, LinkedHashMap

🧠 Lesson Summary

Java’s Map interface defines a structure for storing key-value pairs. HashMap is the most commonly used implementation, offering constant-time performance for basic operations. LinkedHashMap maintains insertion order, while TreeMap keeps keys sorted. Maps are ideal for fast lookups, indexing, and associative data modeling.

🧪 Try It Yourself

// MapDemo.java
import java.util.*;

public class MapDemo {
  public static void main(String[] args) {
    Map scores = new HashMap<>();
    scores.put("Alice", 90);
    scores.put("Bob", 85);
    scores.put("Charlie", 92);

    System.out.println("Bob's score: " + scores.get("Bob"));
    scores.remove("Alice");

    for (String name : scores.keySet()) {
      System.out.println(name + " → " + scores.get(name));
    }
  }
}
