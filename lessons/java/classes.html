🧱 Java Classes & Objects

A class is a blueprint for creating objects. It defines the structure and behavior. An object is an instance of a class — a real-world entity with state and behavior.

📘 What’s the Difference?
  • Class: Defines fields (data) and methods (actions).
  • Object: Created from a class. Each object has its own copy of fields.
