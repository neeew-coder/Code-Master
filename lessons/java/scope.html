Scope in Java defines where a variable can be accessed or modified. Understanding scope helps prevent bugs and makes your code more predictable and secure.

  • Local Scope: Variables declared inside a method or block. Only accessible within that block.
  • Class Scope (Field): Variables declared inside a class but outside methods. Accessible by all methods in the class.
  • Block Scope: Variables declared inside loops, if-statements, etc. Exist only within that block.
