Arrays in Java are fixed-size containers that hold multiple values of the same type. Each element is accessed by its index, starting from 0. Arrays are reference types, meaning they point to memory locations rather than storing values directly. Theyâ€™re ideal for storing structured data like scores, names, or flags, and can be looped through for processing or display.

// ArrayDemo.java
public class ArrayDemo {
  public static void main(String[] args) {
    String[] fruits = {"Apple", "Banana", "Cherry"};

    for (int i = 0; i < fruits.length; i++) {
      System.out.println("Fruit " + i + ": " + fruits[i]);
    }

    // Enhanced for loop
    for (String fruit : fruits) {
      System.out.println("Fruit: " + fruit);
    }
  }
}
