📋 Java List Interface

A List is an ordered collection that allows duplicates. You can access elements by index, insert at specific positions, and iterate in order.

🧠 Lesson Summary

Java’s List interface defines an ordered collection that allows duplicates and indexed access. ArrayList offers fast random access but slower insertions/removals, while LinkedList excels at frequent insertions and deletions. Lists are ideal for managing sequences, queues, and dynamic arrays, and are central to many Java applications.

🧪 Try It Yourself

// ListDemo.java
import java.util.*;

public class ListDemo {
  public static void main(String[] args) {
    List items = new ArrayList<>();
    items.add("Java");
    items.add("Python");
    items.add("C++");

    items.remove("Python");
    items.add(1, "JavaScript");

    for (String item : items) {
      System.out.println(item);
    }
  }
}
