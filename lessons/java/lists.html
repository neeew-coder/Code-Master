📋 Java List Interface

A List is an ordered collection that allows duplicates. You can access elements by index, insert at specific positions, and iterate in order.

📘 Why Use List?
  • Preserves insertion order
  • Allows duplicate elements
  • Supports indexed access and modification
  • Flexible implementations: ArrayList, LinkedList
