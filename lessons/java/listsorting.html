🔢 Sorting Lists in Java

Sorting helps organize data for easier searching, comparison, and presentation. Java provides built-in methods for sorting lists alphabetically, numerically, or by custom rules.

  • How to sort objects by fields like name or age
📘 Sorting Methods
  • Collections.sort(list) — sorts in natural order
  • list.sort(Comparator) — custom sorting
  • Works with ArrayList, LinkedList, and other List types

🧠 Lesson Summary

Java provides built-in tools for sorting lists. Collections.sort() arranges elements in natural order, while Comparator allows custom logic. You can sort strings alphabetically, numbers numerically, or objects by fields. Sorting is essential for search, display, and data organization, and can be reversed or customized for flexibility.

🧪 Try It Yourself

// SortListDemo.java
import java.util.*;

public class SortListDemo {
  public static void main(String[] args) {
    List names = new ArrayList<>();
    names.add("Zara");
    names.add("Alex");
    names.add("John");

    Collections.sort(names); // Ascending
    System.out.println("Ascending: " + names);

    Collections.sort(names, Collections.reverseOrder()); // Descending
    System.out.println("Descending: " + names);
  }
}
