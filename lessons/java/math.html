📐 Java Math

Java provides the Math class for performing mathematical operations like rounding, powers, roots, and trigonometry. All methods in Math are static, so you can call them directly.

📘 What You’ll Learn

  • How to use Math.max(), Math.min(), Math.abs() for comparisons and absolute values
  • How to calculate powers and roots using Math.pow() and Math.sqrt()
  • How to round numbers with Math.round(), Math.ceil(), and Math.floor()
  • How to generate random numbers with Math.random()
  • How trigonometric methods like Math.sin(), Math.cos(), and Math.tan() work

📦 Common Math Methods

  • Math.abs(x) – Absolute value
  • Math.max(a, b) / Math.min(a, b) – Max/Min
  • Math.sqrt(x) – Square root
  • Math.pow(x, y) – x raised to the power y
  • Math.round(x) – Round to nearest integer
  • Math.floor(x) / Math.ceil(x) – Round down/up
  • Math.random() – Random number between 0.0 and 1.0

🧠 Lesson Summary

Java’s Math class provides static methods for performing common mathematical operations. From rounding and comparisons to trigonometry and randomness, these tools help developers write precise, efficient calculations. Understanding how to use Math methods improves numerical accuracy and simplifies complex logic in games, simulations, and data processing.

🧪 Try It Yourself

// MathDemo.java
public class MathDemo {
  public static void main(String[] args) {
    System.out.println("Max: " + Math.max(10, 20));
    System.out.println("Square root: " + Math.sqrt(64));
    System.out.println("Rounded: " + Math.round(5.7));
    System.out.println("Random (0–1): " + Math.random());
    System.out.println("Cosine of 0: " + Math.cos(0));
  }
}
