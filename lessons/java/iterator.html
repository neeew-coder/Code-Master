🔁 Java Iterator

An Iterator provides a way to traverse collections element by element. It works with List, Set, and other Collection types.

🧠 Lesson Summary

Java’s Iterator interface provides a safe way to traverse collections like ArrayList, HashSet, and LinkedList. It supports element removal during iteration and avoids modification errors. Iterators are especially useful when you need fine-grained control over traversal and mutation logic.

🧪 Try It Yourself

// IteratorDemo.java
import java.util.*;

public class IteratorDemo {
  public static void main(String[] args) {
    List names = new ArrayList<>();
    names.add("Alice");
    names.add("Bob");
    names.add("Charlie");

    Iterator iterator = names.iterator();
    while (iterator.hasNext()) {
      String name = iterator.next();
      if (name.equals("Bob")) {
        iterator.remove(); // Safe removal
      }
    }

    System.out.println("Remaining names: " + names);
  }
}
