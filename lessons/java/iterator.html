🔁 Java Iterator

An Iterator provides a way to traverse collections element by element. It works with List, Set, and other Collection types.

📘 Why Use Iterator?
  • Works across different collection types
  • Supports safe removal during iteration
  • Avoids index-based errors
  • Useful for filtering, searching, and modifying collections
