üß† Lesson Summary

Strings in Java are objects backed by a character array and stored in a special memory area called the string pool. They are immutable, meaning once created, their value cannot change. Java provides a rich set of methods for string manipulation, comparison, and formatting. Understanding how strings behave helps prevent memory leaks, optimize performance, and write cleaner code.

// StringDemo.java
public class StringDemo {
  public static void main(String[] args) {
    String name = "Mark";
    System.out.println("Length: " + name.length());
    System.out.println("First letter: " + name.charAt(0));
    System.out.println("Uppercase: " + name.toUpperCase());
    System.out.println("Equals 'mark'? " + name.equals("mark"));
  }
}
