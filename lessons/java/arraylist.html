🧠 Lesson Summary

ArrayList is a part of Java’s Collection framework and provides a dynamic array that grows as needed. Unlike fixed-size arrays, ArrayList allows flexible insertion and deletion of elements. It supports indexed access, iteration, and built-in methods for searching and modifying contents, making it ideal for managing ordered data collections.

🧪 Try It Yourself

// ArrayListDemo.java
import java.util.ArrayList;

public class ArrayListDemo {
  public static void main(String[] args) {
    ArrayList colors = new ArrayList<>();
    colors.add("Red");
    colors.add("Green");
    colors.add("Blue");

    colors.remove("Green");
    colors.add(1, "Yellow");

    for (String color : colors) {
      System.out.println(color);
    }

    System.out.println("Contains Blue? " + colors.contains("Blue"));
    System.out.println("Size: " + colors.size());
  }
}
