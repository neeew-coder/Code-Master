Java Variables

Variables are containers for storing data values. In Java, each variable must be declared with a specific type.

📦 Declaring Variables

To declare a variable, specify the type and name: 

int myNum = 5;
String myText = "Hello";

🔤 Variable Types

Java is a statically typed language, so you must declare the type of variable:

  • int – stores integers (whole numbers)
  • double – stores decimal numbers
  • char – stores single characters
  • String – stores text
  • boolean – stores true or false
