ðŸ§  Lesson Summary

The switch statement in Java offers a clean way to handle multiple conditions based on a single variable. Each case matches a value, and break prevents unintended execution of subsequent cases. default handles unmatched values. Switch is ideal for discrete, predictable branching and improves readability over long if-else chains.

ðŸ§ª Try It Yourself

// SwitchDemo.java
public class SwitchDemo {
  public static void main(String[] args) {
    int day = 3;

    switch (day) {
      case 1:
        System.out.println("Monday");
        break;
      case 2:
        System.out.println("Tuesday");
        break;
      case 3:
        System.out.println("Wednesday");
        break;
      default:
        System.out.println("Invalid day");
    }
  }
}
