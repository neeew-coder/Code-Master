Java Switch Statement

The switch statement is a cleaner alternative to multiple if...else if conditions. It compares a single value against multiple cases and executes matching code blocks.

📚 Explanation

  • switch(day) checks the value of day.
  • case blocks match specific values (e.g., case 3).
  • break exits the switch after a match — without it, execution "falls through" to the next case.
  • default runs if no case matches — like an else.
  • Switch is ideal for clean, readable branching when comparing one variable to many values.

🧠 Extra Insight

Java’s switch works with int, char, String, and enum types. Starting with Java 14+, you can use switch expressions for more concise logic: 

String result = switch (day) {
  case 1 -> "Monday";
  case 2 -> "Tuesday";
  case 3 -> "Wednesday";
  default -> "Unknown";
};
System.out.println(result);
