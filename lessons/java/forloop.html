ğŸ“š Explanation

  • for loops have three parts: initialization, condition, and update.
  • Initialization runs once at the start (e.g. int i = 1).
  • Condition is checked before each iteration (e.g. i <= 5).
  • Update runs after each iteration (e.g. i++).
  • Loop stops when the condition becomes false.

// ForLoopDemo.java
public class ForLoopDemo {
  public static void main(String[] args) {
    for (int i = 1; i <= 5; i++) {
      System.out.println("Iteration: " + i);
    }

    // Nested loop example
    for (int row = 1; row <= 3; row++) {
      for (int col = 1; col <= 3; col++) {
        System.out.print("* ");
      }
      System.out.println();
    }
  }
}
