📦 Java Packages & APIs

Packages group related classes together, making code modular and easier to manage. APIs (Application Programming Interfaces) are libraries of pre-written code that provide useful functionality — like working with dates, files, or networks.

📘 What Are Packages & APIs?
  • Package: A namespace that organizes classes (e.g., java.util, java.io)
  • API: A set of classes and methods provided by Java or third parties to perform common tasks
  • Use import to bring packages into your code
