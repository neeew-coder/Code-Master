ğŸ§  Lesson Summary

In Java, break is used to exit a loop or switch block prematurely, while continue skips the rest of the current iteration and proceeds to the next. These control statements help manage loop behavior, especially when filtering data or handling edge cases. Used wisely, they simplify logic and improve performance â€” but overuse can reduce readability.

// BreakContinueDemo.java
public class BreakContinueDemo {
  public static void main(String[] args) {
    for (int i = 1; i <= 10; i++) {
      if (i == 7) break;
      if (i % 2 == 0) continue;
      System.out.println("Odd: " + i);
    }
  }
}
