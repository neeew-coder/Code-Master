🔘 Java Boolean

A boolean in Java represents one of two values: true or false. Booleans are commonly used in conditions, loops, and logical operations.

📦 Boolean Basics

  • boolean isActive = true;
  • Booleans are often returned by comparison operators: 5 > 3 → true
  • Logical operators: && (AND), || (OR), ! (NOT)
