🔘 Java Boolean

A boolean in Java represents one of two values: true or false. Booleans are commonly used in conditions, loops, and logical operations.

🧠 Lesson Summary

In Java, the boolean type represents logical values: true or false. It’s essential for controlling program flow, making decisions, and evaluating conditions. Booleans are often used with comparison and logical operators to determine whether blocks of code should execute. Mastering boolean logic is key to writing clean, predictable, and responsive Java programs.

🧪 Try It Yourself

// BooleanDemo.java
public class BooleanDemo {
  public static void main(String[] args) {
    boolean isLoggedIn = true;
    boolean hasPermission = false;

    if (isLoggedIn && hasPermission) {
      System.out.println("Access granted.");
    } else {
      System.out.println("Access denied.");
    }
  }
}
